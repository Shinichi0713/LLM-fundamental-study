# 順序集合

## 順序集合

**順序集合（ordered set）**は、集合の要素の間に「大小関係」や「前後関係」のような**順序**を定めた構造です。  
数学の多くの分野で基本的な役割を果たします。

### 1. 順序集合の数学的定義

__(1) 二項関係としての定義__

集合 $X$ 上の**二項関係** $\leq$ が次の3条件を満たすとき、$(X,\leq)$ を**前順序集合（preordered set）**といいます。

1. **反射律**：任意の $x \in X$ に対し $x \leq x$。
2. **推移律**：任意の $x,y,z \in X$ に対し、$x \leq y$ かつ $y \leq z$ ならば $x \leq z$。

さらに、次の条件を満たすとき、$(X,\leq)$ を**半順序集合（partially ordered set, poset）**といいます。

3. **反対称律**：任意の $x,y \in X$ に対し、$x \leq y$ かつ $y \leq x$ ならば $x = y$。

さらに、次の条件を満たすとき、$(X,\leq)$ を**全順序集合（totally ordered set / linearly ordered set）**といいます。

4. **全順序性**：任意の $x,y \in X$ に対し、$x \leq y$ または $y \leq x$ の少なくとも一方が成り立つ。

まとめると：

- **前順序集合**：反射律＋推移律
- **半順序集合**：前順序＋反対称律
- **全順序集合**：半順序＋全順序性

__(2) 記号と用語__

- $\leq$ を**順序関係（order relation）**と呼ぶ。
- $x \leq y$ を「$x$ は $y$ 以下」「$y$ は $x$ 以上」などと読む。
- $x \leq y$ かつ $x \neq y$ のとき、$x < y$ と書くこともある（**狭義の順序**）。
- 順序集合 $(X,\leq)$ を単に $X$ と書くことも多い。

### 2. 具体例

__(1) 数の集合__

- $(\mathbb{R}, \leq)$：実数全体に通常の大小関係を入れた**全順序集合**。
- $(\mathbb{N}, \leq)$：自然数全体も全順序集合。
- $(\mathbb{Q}, \leq)$：有理数全体も全順序集合。

__(2) べき集合（部分集合の包含）__

- $X$ を任意の集合とし、$\mathcal{P}(X)$ をそのべき集合とする。
- 包含関係 $\subset$ を順序とすると、$(\mathcal{P}(X), \subset)$ は**半順序集合**（一般には全順序ではない）。
  - 例：$X = \{1,2\}$ のとき、$\{1\}$ と $\{2\}$ は比較不能（どちらも他方の部分集合ではない）。

__(3) 整除関係__

- $(\mathbb{N}, \mid)$：自然数全体に「$a \mid b$（$a$ は $b$ を割り切る）」という関係を入れると、半順序集合になる。
  - 反射律：$a \mid a$
  - 推移律：$a \mid b$ かつ $b \mid c$ なら $a \mid c$
  - 反対称律：$a \mid b$ かつ $b \mid a$ なら $a = b$
  - 全順序ではない（例：2 と 3 は比較不能）。

### 3. なぜ順序集合が必要なのか

__(1) 「大小」や「前後」を数学的に扱うため__

- 実数や自然数の大小関係、時間の前後関係、集合の包含関係など、現実世界や数学の多くの場面で「順序」が自然に現れます。
- 順序集合は、こうした**順序構造を抽象化**したものです。
- これにより、「最大元・最小元」「上界・下界」「極大元・極小元」など、順序に基づく概念を一般の集合に対して定義できます。

__(2) 極限・連続性・完備性の基礎__

- 実数直線 $(\mathbb{R},\leq)$ は**全順序かつ完備**（任意の有界な部分集合が上限を持つ）であり、解析学の基礎となります。
- 数列の極限、関数の連続性、微分・積分などは、実数の順序構造と位相構造に強く依存しています。
- 順序集合の一般論（完備半順序集合、dcpoなど）は、領域理論やプログラム意味論においても重要です。

__(3) 代数的構造との組み合わせ__

- **束（lattice）**：任意の2元が上限（join）と下限（meet）を持つ半順序集合。ブール代数、ハッセ図などに現れる。
- **整列集合（well-ordered set）**：任意の空でない部分集合が最小元を持つ全順序集合。整列可能定理や順序数理論の基礎。
- **順序群・順序環**：群や環の演算と整合的な順序構造を持つもの。実数体 $(\mathbb{R},+,\cdot,\leq)$ はその典型例。

__(4) 集合論・論理・計算機科学での応用__

- **集合論**：順序数は整列集合の同型類として定義され、無限の大きさを測る基本的な道具です。
- **モデル理論**：構造に順序が入っているかどうかで、満たす論理式のクラスが変わります。
- **計算機科学**：プログラムの実行順序、データ構造（ヒープ、二分探索木など）、型システムの部分型関係など、多くの場面で順序集合が現れます。

### 4. 順序集合の基本的な概念（概要）

順序集合を扱う上で重要な概念をいくつか挙げます（詳細は別途説明可能です）。

- **最大元・最小元**：集合全体の中で最も大きい／小さい元。
- **極大元・極小元**：それより大きい／小さい元が存在しない元。
- **上界・下界**：与えられた部分集合を「上から／下から」抑える元。
- **上限（sup）・下限（inf）**：上界／下界の集合の最小元／最大元。
- **鎖（chain）**：全順序な部分集合。
- **反鎖（antichain）**：互いに比較不能な元からなる部分集合。
- **整列集合**：任意の空でない部分集合が最小元を持つ全順序集合。

## 部分順序集合

**部分順序集合（partially ordered set, poset）**は、集合の要素の間に「順序」を定めた構造のうち、**反射律・推移律・反対称律**を満たすものを指します。  
「全順序」とは異なり、**すべての2元が比較可能とは限らない**という点が特徴です。

### 1. 部分順序集合の定義

集合 $X$ 上の**二項関係** $\leq$ が次の3条件を満たすとき、$(X,\leq)$ を**部分順序集合（poset）**といいます。

1. **反射律**：任意の $x \in X$ に対し $x \leq x$。
2. **推移律**：任意の $x,y,z \in X$ に対し、$x \leq y$ かつ $y \leq z$ ならば $x \leq z$。
3. **反対称律**：任意の $x,y \in X$ に対し、$x \leq y$ かつ $y \leq x$ ならば $x = y$。

このとき、$\leq$ を**部分順序関係（partial order）**と呼びます。

__記号と用語__

- $x \leq y$：  
  - 「$x$ は $y$ 以下」「$y$ は $x$ 以上」などと読む。
- $x < y$：  
  - $x \leq y$ かつ $x \neq y$ のとき（**狭義の順序**）。
- $x$ と $y$ が**比較可能（comparable）**：  
  - $x \leq y$ または $y \leq x$ の少なくとも一方が成り立つとき。
- $x$ と $y$ が**比較不能（incomparable）**：  
  - $x \leq y$ も $y \leq x$ も成り立たないとき。

### 2. 部分順序集合の具体例

__(1) 数の集合（通常の大小）__

- $(\mathbb{R}, \leq)$：実数全体に通常の大小関係を入れると、**全順序集合**（したがって部分順序集合でもある）。
- $(\mathbb{N}, \leq)$：自然数全体も同様。

これらは**すべての2元が比較可能**なので、特に**全順序集合**と呼ばれます。

__(2) べき集合（部分集合の包含）__

- $X$ を任意の集合とし、$\mathcal{P}(X)$ をそのべき集合とする。
- 包含関係 $\subset$ を順序とすると、$(\mathcal{P}(X), \subset)$ は**部分順序集合**。
  - 反射律：$A \subset A$
  - 推移律：$A \subset B$ かつ $B \subset C$ なら $A \subset C$
  - 反対称律：$A \subset B$ かつ $B \subset A$ なら $A = B$

例：$X = \{1,2\}$ のとき、
- $\{1\} \subset \{1,2\}$、$\{2\} \subset \{1,2\}$ は成り立つが、
- $\{1\}$ と $\{2\}$ は互いに包含関係になく、**比較不能**です。

__(3) 整除関係__

- $(\mathbb{N}, \mid)$：自然数全体に「$a \mid b$（$a$ は $b$ を割り切る）」という関係を入れると、部分順序集合になる。
  - 反射律：$a \mid a$
  - 推移律：$a \mid b$ かつ $b \mid c$ なら $a \mid c$
  - 反対称律：$a \mid b$ かつ $b \mid a$ なら $a = b$

例：
- $2 \mid 4$、$3 \mid 6$ は成り立つが、
- $2$ と $3$ は互いに割り切れないので**比較不能**。

__(4) ベクトル空間の部分空間__

- $V$ をベクトル空間とし、その部分空間全体の集合を考える。
- 包含関係 $\subset$ を順序とすると、部分順序集合になる。
- 例：$\mathbb{R}^2$ の部分空間（原点、直線、平面全体）は包含関係で順序づけられる。

### 3. 部分順序集合の基本的な概念

部分順序集合には、順序に基づくさまざまな概念が定義されます。

__(1) 最大元・最小元__

- **最大元（maximum element）**：  
  すべての $x \in X$ に対して $x \leq m$ となる $m \in X$。
- **最小元（minimum element）**：  
  すべての $x \in X$ に対して $m \leq x$ となる $m \in X$。

存在すれば一意です（反対称律より）。

__(2) 極大元・極小元__

- **極大元（maximal element）**：  
  $m \in X$ であって、$m < x$ となる $x \in X$ が存在しないもの。
- **極小元（minimal element）**：  
  $m \in X$ であって、$x < m$ となる $x \in X$ が存在しないもの。

最大元・最小元とは異なり、**複数存在し得る**点が重要です。

__(3) 上界・下界・上限・下限__

部分集合 $A \subset X$ を考えます。

- **上界（upper bound）**：  
  すべての $a \in A$ に対して $a \leq u$ となる $u \in X$。
- **下界（lower bound）**：  
  すべての $a \in A$ に対して $\ell \leq a$ となる $\ell \in X$。
- **上限（supremum, least upper bound）**：  
  上界全体の集合の**最小元**（存在すれば一意）。
- **下限（infimum, greatest lower bound）**：  
  下界全体の集合の**最大元**（存在すれば一意）。

例：$(\mathbb{R},\leq)$ では、有界な集合は必ず上限・下限を持ちます（実数の完備性）。

__(4) 鎖（chain）と反鎖（antichain）__

- **鎖（chain）**：  
  $X$ の部分集合 $C$ で、任意の2元が比較可能なもの（すなわち $C$ 上で $\leq$ が全順序になる）。
- **反鎖（antichain）**：  
  $X$ の部分集合 $A$ で、任意の相異なる2元が比較不能なもの。

例：べき集合 $\mathcal{P}(\{1,2,3\})$ では、
- $\{\emptyset, \{1\}, \{1,2\}, \{1,2,3\}\}$ は鎖、
- $\{\{1\}, \{2\}, \{3\}\}$ は反鎖です。

### 4. なぜ部分順序集合が重要なのか

__(1) 「比較不能」を許容する現実的なモデル__

- 実数の大小のように「すべての2元が比較可能」な状況は特殊です。
- 多くの現実的な状況（集合の包含、整除関係、部分型関係など）では、**比較不能な要素**が自然に現れます。
- 部分順序集合は、こうした**部分的にしか順序づけられない構造**を数学的に扱うための枠組みです。

__(2) 束（lattice）・ブール代数の基礎__

- 任意の2元が上限と下限を持つ部分順序集合を**束（lattice）**といいます。
- さらに補元演算などが備わると**ブール代数**となり、論理・集合演算・電子回路の設計などに応用されます。
- これらはすべて部分順序集合を土台としています。

__(3) 順序理論・組合せ論・計算機科学__

- **Dilworthの定理**や**Mirskyの定理**など、鎖と反鎖の関係に関する重要な結果があります。
- プログラムの実行順序、データ構造（ヒープ、優先度付きキューなど）、型システムの部分型関係など、計算機科学の多くの概念が部分順序集合としてモデル化されます。

## 順序同型、順序型

**順序同型（order isomorphism）**と**順序型（order type）**は、順序集合の「形」や「構造」を比較・分類するための概念です。  
集合の「対等」（全単射）が「要素の個数」を比較するのに対し、順序同型は**順序関係まで含めた構造**を比較します。

### 1. 順序同型（order isomorphism）

__定義__

2つの順序集合 $(X,\leq_X)$ と $(Y,\leq_Y)$ について、写像 $f: X \to Y$ が次の条件を満たすとき、**順序同型写像（order isomorphism）**であるといいます。

1. **全単射**である：  
   - 単射かつ全射。
2. **順序を保つ（order-preserving）**：  
   - 任意の $x_1, x_2 \in X$ に対し、  
     $$
     x_1 \leq_X x_2 \iff f(x_1) \leq_Y f(x_2)
     $$

このとき、$(X,\leq_X)$ と $(Y,\leq_Y)$ は**順序同型（order isomorphic）**であるといい、
$$
(X,\leq_X) \cong (Y,\leq_Y)
$$
と書きます。

__直感的な意味__

- 順序同型写像は、**要素の対応づけだけでなく、順序関係も完全に保存する**全単射です。
- つまり、2つの順序集合は「順序構造まで含めて同じ形」をしている、とみなせます。
- 例：自然数全体 $(\mathbb{N},\leq)$ と正の偶数全体 $(\{2,4,6,\dots\},\leq)$ は順序同型（写像 $n \mapsto 2n$）。

__具体例__

1. **有限全順序集合**  
   - $(\{1,2,3\},\leq)$ と $(\{a,b,c\},\leq)$（辞書順など）は順序同型。
   - 全単射 $1 \mapsto a,\ 2 \mapsto b,\ 3 \mapsto c$ は順序を保つ。

2. **可算無限の全順序集合**  
   - $(\mathbb{N},\leq)$ と $(\mathbb{Z},\leq)$ は**集合としては対等**だが、**順序同型ではない**。  
     - $\mathbb{Z}$ には最小元がないが、$\mathbb{N}$ には最小元 1 があるため。

3. **べき集合の包含順序**  
   - $(\mathcal{P}(\{1,2\}),\subset)$ と $(\{0,1,2,3\},\leq)$ は順序同型ではない（構造が異なる）。

### 2. 順序型（order type）

__定義__

順序集合 $(X,\leq)$ の**順序型（order type）**とは、その順序構造を**同型類として捉えたもの**です。

より正確には：

- 順序同型という同値関係（反射律・対称律・推移律を満たす）によって、すべての順序集合を類別します。
- この同値類のそれぞれを**順序型**と呼びます。

記号では、順序型を $\operatorname{ot}(X,\leq)$ や単に $\operatorname{type}(X)$ などと書くことがあります。

__直感的な意味__

- 順序型は、順序集合の「形」や「パターン」を表すラベルのようなものです。
- 順序同型な順序集合は**同じ順序型**を持ち、順序同型でないものは**異なる順序型**を持ちます。

__具体例（全順序集合の場合）__

全順序集合の順序型は特に重要で、**順序数（ordinal number）**と深く関係します。

1. **有限順序型**  
   - 要素数 $n$ の全順序集合の順序型は、自然数 $n$ で表されることが多い。  
   - 例：$\operatorname{ot}(\{1,2,3\},\leq) = 3$。

2. **可算無限の順序型**  
   - 自然数全体 $(\mathbb{N},\leq)$ の順序型は $\omega$（オメガ）と呼ばれる。  
   - 例：$\{1,2,3,\dots\}$ や $\{2,4,6,\dots\}$ はどちらも順序型 $\omega$。

3. **整数全体の順序型**  
   - $(\mathbb{Z},\leq)$ の順序型は $\zeta$（ゼータ）などと書かれることがある。  
   - 最小元がなく、両方向に無限に伸びる構造。

4. **有理数全体の順序型**  
   - $(\mathbb{Q},\leq)$ の順序型は $\eta$（エータ）と呼ばれる。  
   - 「稠密（dense）かつ可算」という特徴的な構造を持つ。

5. **実数全体の順序型**  
   - $(\mathbb{R},\leq)$ の順序型は $\lambda$ などと書かれることもあるが、標準的な記号はない。  
   - 連続体濃度を持ち、完備な全順序集合。

### 3. 順序同型と順序型の性質

__(1) 順序同型は同値関係__

- 反射律：恒等写像は順序同型。
- 対称律：$f$ が順序同型なら逆写像 $f^{-1}$ も順序同型。
- 推移律：$f: X \to Y$ と $g: Y \to Z$ が順序同型なら、合成 $g \circ f$ も順序同型。

したがって、順序同型は**同値関係**であり、順序型はその同値類です。

__(2) 順序同型なら集合としても対等__

- 順序同型写像は全単射なので、**順序同型な順序集合は必ず対等（同じ濃度）**です。
- 逆は成り立ちません（例：$\mathbb{N}$ と $\mathbb{Z}$ は対等だが順序同型ではない）。

__(3) 順序型と濃度__

- 順序型は**順序構造**を、濃度は**集合の大きさ**を表します。
- 同じ順序型なら同じ濃度ですが、同じ濃度でも異なる順序型を持つことがあります。
- 例：  
  - $\mathbb{N}$ と $\mathbb{Z}$ はどちらも濃度 $\aleph_0$ だが、順序型は異なる（$\omega$ と $\zeta$）。

### 4. なぜ順序同型・順序型が必要なのか

__(1) 順序構造の「形」を比較するため__

- 集合の対等（全単射）だけでは、「最小元があるか」「稠密か」「完備か」といった**順序的な性質**は保存されません。
- 順序同型は、これらの性質まで含めて「同じ形か」を判定するための概念です。

__(2) 順序数理論の基礎__

- 整列集合（well-ordered set）の順序型は**順序数（ordinal number）**と呼ばれ、無限の大きさを測る基本的な道具です。
- 順序数は、整列集合の順序同型類として定義されます。

__(3) モデル理論・記述集合論での応用__

- 与えられた順序型を持つ構造がどのような論理式を満たすか、という問題はモデル理論で重要です。
- 実数直線や有理数直線の順序型は、記述集合論や位相群論でも頻繁に現れます。

## 全順序集合

**全順序集合（totally ordered set / linearly ordered set）** は、集合の要素の間に「順序」が定まっており、**任意の2つの要素が必ず比較可能**であるような順序集合です。  
数学的には、部分順序集合の定義に「全順序性」を加えたものとして定義されます。

### 1. 全順序集合の定義

集合 $X$ 上の**二項関係** $\leq$ が次の4条件を満たすとき、$(X,\leq)$ を**全順序集合**といいます。

1. **反射律**：任意の $x \in X$ に対し $x \leq x$。
2. **推移律**：任意の $x,y,z \in X$ に対し、$x \leq y$ かつ $y \leq z$ ならば $x \leq z$。
3. **反対称律**：任意の $x,y \in X$ に対し、$x \leq y$ かつ $y \leq x$ ならば $x = y$。
4. **全順序性（totality / linearity）**：  
   任意の $x,y \in X$ に対し、$x \leq y$ または $y \leq x$ の少なくとも一方が成り立つ。

このとき、$\leq$ を**全順序関係（total order / linear order）** と呼びます。

__記号と用語__

- $x \leq y$：  
  - 「$x$ は $y$ 以下」「$y$ は $x$ 以上」などと読む。
- $x < y$：  
  - $x \leq y$ かつ $x \neq y$ のとき（**狭義の順序**）。
- **線形順序（linear order）**：  
  - 全順序と同じ意味で使われることが多い。要素が「一直線上に並ぶ」イメージ。

### 2. 全順序集合の具体例

__(1) 数の集合（通常の大小）__

- $(\mathbb{R}, \leq)$：実数全体に通常の大小関係を入れたもの。  
  → 任意の2実数は大小比較可能なので全順序集合。
- $(\mathbb{Q}, \leq)$：有理数全体も同様。
- $(\mathbb{N}, \leq)$：自然数全体も全順序集合。
- $(\mathbb{Z}, \leq)$：整数全体も全順序集合。

これらはすべて、**数直線**のように要素が一直線上に並ぶ構造を持ちます。

__(2) 辞書式順序（lexicographic order）__

- 有限列や文字列に「辞書順」を入れると全順序集合になります。
- 例：アルファベット順に並べた単語の集合（"apple" < "banana" など）。

__(3) 順序数（ordinal numbers）__

- 順序数は**整列集合**（後述）であり、特に全順序集合です。
- 例：$\omega$（自然数全体の順序型）、$\omega+1$、$\omega\cdot2$ など。

### 3. 全順序集合と部分順序集合の違い

- **部分順序集合（poset）**：  
  反射律・推移律・反対称律を満たすが、**比較不能な要素**が存在し得る。
- **全順序集合**：  
  部分順序集合の条件に加えて、**任意の2元が比較可能**。

例：
- べき集合 $(\mathcal{P}(\{1,2\}),\subset)$ は部分順序集合だが、全順序ではない  
  （$\{1\}$ と $\{2\}$ は比較不能）。
- $(\mathbb{N},\leq)$ は全順序集合（任意の2自然数は大小比較可能）。

### 4. 全順序集合の基本的な概念

全順序集合では、部分順序集合と同じ概念（最大元・最小元、上界・下界など）が定義されますが、**全順序性により性質が強まる**ことが多いです。

__(1) 最大元・最小元__

- 全順序集合では、最大元・最小元が存在すれば一意です。
- 例：$(\mathbb{N},\leq)$ の最小元は 1、最大元は存在しない。

__(2) 上界・下界・上限・下限__

- 全順序集合では、上界・下界の構造が比較的単純になります。
- 特に**実数直線** $(\mathbb{R},\leq)$ では、有界な部分集合は必ず上限・下限を持ちます（実数の完備性）。

__(3) 鎖（chain）__

- 全順序集合そのものが**鎖**です（任意の2元が比較可能）。
- 部分順序集合の中の「全順序な部分集合」を鎖と呼ぶことがありますが、全順序集合では全体が鎖です。

### 5. 整列集合（well-ordered set）

全順序集合のうち、特に重要なサブクラスが**整列集合**です。

__定義__

全順序集合 $(X,\leq)$ が**整列集合**であるとは、  
任意の空でない部分集合 $A \subset X$ が**最小元**を持つことをいいます。

- 例：$(\mathbb{N},\leq)$ は整列集合（任意の空でない自然数の集合には最小元がある）。
- 例：$(\mathbb{Z},\leq)$ は整列集合ではない（負の整数全体には最小元がない）。
- 例：$(\mathbb{Q},\leq)$ も整列集合ではない（開区間などに最小元がない）。

__順序数との関係__

- 整列集合の順序同型類を**順序数（ordinal number）** と呼びます。
- 順序数は無限の大きさを測る基本的な道具であり、集合論の基盤となります。

### 6. なぜ全順序集合が必要なのか

__(1) 「一直線に並ぶ」構造をモデル化するため__

- 数直線、時間の流れ、辞書順、優先順位など、現実世界の多くの「順序」は全順序です。
- 全順序集合は、こうした**線形な順序構造**を数学的に扱うための枠組みです。

__(2) 解析学の基礎__

- 実数直線 $(\mathbb{R},\leq)$ は全順序かつ完備であり、数列の極限、関数の連続性、微分・積分など、解析学のほぼすべての概念がこの構造に依存しています。

__(3) 整列可能定理と選択公理__

- **整列可能定理**（任意の集合に整列順序が存在する）は選択公理と同値であり、集合論の重要な結果です。
- これにより、任意の集合を「一直線に並べる」ことができることが保証されます（ただし具体的な順序は明示されない）。

__(4) 順序数理論・超限帰納法__

- 順序数は整列集合の順序型として定義され、超限帰納法・超限再帰の基礎となります。
- これは数学の証明手法や再帰的定義を無限に拡張するための強力な道具です。


## 最大元、最小元、極大元、極小元、上限、下限

**最大元・最小元・極大元・極小元・上限・下限**は、順序集合（特に部分順序集合）において、部分集合の「端」や「境界」を表す概念です。  
以下、順序集合 $(X,\leq)$ とその部分集合 $A \subset X$ に対して定義します。

### 1. 最大元（maximum element）と最小元（minimum element）

__最大元の定義__

元 $m \in A$ が $A$ の**最大元**であるとは、  
すべての $a \in A$ に対して $a \leq m$ が成り立つことをいいます。

- 記号：$\max A$ などと書く。
- 意味：$A$ の要素の中で「最も大きい」元。

__最小元の定義__

元 $m \in A$ が $A$ の**最小元**であるとは、  
すべての $a \in A$ に対して $m \leq a$ が成り立つことをいいます。

- 記号：$\min A$ などと書く。
- 意味：$A$ の要素の中で「最も小さい」元。

__性質__

- 存在すれば**一意**です（反対称律より）。
- 例：
  - $A = \{1,2,3\}$（通常の大小）なら、$\min A = 1,\ \max A = 3$。
  - $A = (0,1) \subset \mathbb{R}$ なら、最小元も最大元も存在しない。

### 2. 極大元（maximal element）と極小元（minimal element）

__極大元の定義__

元 $m \in A$ が $A$ の**極大元**であるとは、  
「$m < a$ となる $a \in A$ が存在しない」ことをいいます。

- 言い換え：$m \leq a$ となる $a \in A$ は $m$ 自身に限られる。
- 意味：$A$ の中で「それより大きい元がない」元。

__極小元の定義__

元 $m \in A$ が $A$ の**極小元**であるとは、  
「$a < m$ となる $a \in A$ が存在しない」ことをいいます。

- 言い換え：$a \leq m$ となる $a \in A$ は $m$ 自身に限られる。
- 意味：$A$ の中で「それより小さい元がない」元。

__最大元・最小元との違い__

- **最大元**は「すべての元より大きい」、**極大元**は「それより大きい元がない」。
- 最大元は極大元ですが、逆は必ずしも成り立ちません。
- 例：$A = \{1,2,3,4\}$ で、順序を「整除関係 $a \mid b$」とする。
  - 極大元：3, 4（これらより大きい元はない）。
  - 最大元：存在しない（3 と 4 は比較不能）。

### 3. 上界（upper bound）と下界（lower bound）

__上界の定義__

元 $u \in X$ が $A$ の**上界**であるとは、  
すべての $a \in A$ に対して $a \leq u$ が成り立つことをいいます。

- 意味：$A$ の要素を「上から抑える」元。
- 例：$A = (0,1) \subset \mathbb{R}$ の上界は、1 以上のすべての実数。

__下界の定義__

元 $\ell \in X$ が $A$ の**下界**であるとは、  
すべての $a \in A$ に対して $\ell \leq a$ が成り立つことをいいます。

- 意味：$A$ の要素を「下から支える」元。
- 例：$A = (0,1) \subset \mathbb{R}$ の下界は、0 以下のすべての実数。

__性質__

- 上界・下界は $A$ の外の元でも構いません。
- 上界の集合を $U(A)$、下界の集合を $L(A)$ と書くことがあります。

### 4. 上限（supremum）と下限（infimum）

__上限の定義__

$A$ の上界全体の集合 $U(A)$ が**最小元**を持つとき、その元を $A$ の**上限**といいます。

- 記号：$\sup A$（supremum）。
- 意味：上界の中で「最も小さい」元。
- 存在すれば**一意**です。

__下限の定義__

$A$ の下界全体の集合 $L(A)$ が**最大元**を持つとき、その元を $A$ の**下限**といいます。

- 記号：$\inf A$（infimum）。
- 意味：下界の中で「最も大きい」元。
- 存在すれば**一意**です。

__性質__

- $\sup A$ は「最小上界（least upper bound）」、$\inf A$ は「最大下界（greatest lower bound）」とも呼ばれます。
- 例：
  - $A = (0,1) \subset \mathbb{R}$ なら、$\inf A = 0,\ \sup A = 1$。
  - $A = \{1/n \mid n \in \mathbb{N}\} \subset \mathbb{R}$ なら、$\inf A = 0,\ \sup A = 1$（最大元は 1）。

### 5. 最大元・最小元と上限・下限の関係

- 最大元が存在すれば、それは上限と一致します（$\max A = \sup A$）。
- 最小元が存在すれば、それは下限と一致します（$\min A = \inf A$）。
- 逆は成り立ちません：
  - 例：$A = (0,1)$ では $\sup A = 1$ だが、最大元は存在しない。

### 6. 具体例での確認

__例1：実数の区間__

- $A = [0,1]$：  
  - $\min A = 0,\ \max A = 1$。  
  - $\inf A = 0,\ \sup A = 1$。
- $A = (0,1)$：  
  - 最小元・最大元は存在しない。  
  - $\inf A = 0,\ \sup A = 1$。

__例2：べき集合の包含順序__

$X = \{1,2,3\}$、$\mathcal{P}(X)$ に包含関係 $\subset$ を入れる。

- $A = \{\{1\},\{2\},\{3\}\}$：  
  - 極大元：$\{1\},\{2\},\{3\}$（いずれも包含関係で比較不能）。  
  - 最大元：存在しない。  
  - 上界：$\{1,2,3\}$ のみ。  
  - $\sup A = \{1,2,3\}$。

__例3：整除関係__

$(\mathbb{N},\mid)$：$a \mid b$ を順序とする。

- $A = \{2,3,4,6\}$：  
  - 極大元：3, 4, 6（これらより大きい元はない）。  
  - 最大元：存在しない（3 と 4 は比較不能）。  
  - 上界：12, 24, …（公倍数）。  
  - $\sup A = \mathrm{lcm}(2,3,4,6) = 12$。

## 実数の連続性

**実数の連続性（continuity of the real numbers）** は、実数全体 $\mathbb{R}$ が「数直線として隙間なくつながっている」ことを保証する性質です。  
数学的には、**実数の完備性（completeness）** とも呼ばれ、いくつか同値な形で表現されます（上限公理・区間縮小法・有界単調数列の収束など）。

以下、主な定式化とその意味を説明します。

### 1. 上限公理（least upper bound property）

__定義__

実数全体 $\mathbb{R}$ は次の性質を持ちます：

> 任意の空でない上に有界な部分集合 $A \subset \mathbb{R}$ は、**上限（最小上界）**を持つ。

すなわち、$A \neq \varnothing$ かつある $M \in \mathbb{R}$ が存在してすべての $a \in A$ について $a \leq M$ ならば、  
ある実数 $s \in \mathbb{R}$ が存在して：
- すべての $a \in A$ について $a \leq s$（$s$ は上界）
- 任意の上界 $u$ について $s \leq u$（$s$ は最小の上界）

この $s$ を $\sup A$ と書きます。

__具体例__

- $A = (0,1)$：上界は 1 以上の実数。最小上界は 1 → $\sup A = 1$。
- $A = \{1/n \mid n \in \mathbb{N}\}$：上界は 1 以上の実数。最小上界は 1 → $\sup A = 1$。
- $A = \{x \in \mathbb{Q} \mid x^2 < 2\}$：有理数の中では最小上界が存在しないが、実数では $\sup A = \sqrt{2}$。

この最後の例が、**有理数 $\mathbb{Q}$ が連続でない（完備でない）** ことの典型です。

### 2. 下限公理（greatest lower bound property）

上限公理と双対的に、次も成り立ちます：

> 任意の空でない下に有界な部分集合 $A \subset \mathbb{R}$ は、**下限（最大下界）** を持つ。

記号では $\inf A$ と書きます。

### 3. 有界単調数列の収束（monotone convergence theorem）

__定理__

実数列 $\{a_n\}$ が次の条件を満たすとする：

- **単調増加**：$a_1 \leq a_2 \leq a_3 \leq \dots$
- **上に有界**：ある $M \in \mathbb{R}$ が存在してすべての $n$ について $a_n \leq M$

このとき、$\{a_n\}$ は実数に収束する。  
より正確には、極限 $\lim_{n\to\infty} a_n$ が存在し、それは $\sup\{a_n \mid n \in \mathbb{N}\}$ に等しい。

同様に、単調減少で下に有界な数列も収束します。

__例__

- $a_n = 1 - 1/n$：単調増加、上に有界（1 以下）→ $\lim a_n = 1$。
- $a_n = \sum_{k=1}^n 1/k^2$：単調増加、上に有界（$\pi^2/6$ 以下）→ 収束。

### 4. 区間縮小法（nested interval property）

__定理__

閉区間の列 $\{I_n = [a_n, b_n]\}$ が次の条件を満たすとする：

- **縮小**：$I_1 \supset I_2 \supset I_3 \supset \dots$
- **長さが 0 に収束**：$\lim_{n\to\infty} (b_n - a_n) = 0$

このとき、すべての区間 $I_n$ に属する実数が**ちょうど1つ**存在する。  
すなわち、$\bigcap_{n=1}^\infty I_n$ は1点からなる。

__例__

- $I_n = [0, 1/n]$：$\bigcap I_n = \{0\}$。
- 有理数では成り立たない（例：$[ \sqrt{2}-1/n, \sqrt{2}+1/n ] \cap \mathbb{Q}$ の共通部分は空）。

### 5. Cauchy列の収束（Cauchy completeness）

__定義__

実数列 $\{a_n\}$ が**Cauchy列**であるとは：

> 任意の $\varepsilon > 0$ に対し、ある $N \in \mathbb{N}$ が存在して、  
> すべての $m,n \geq N$ について $|a_n - a_m| < \varepsilon$ が成り立つ。

すなわち、「十分先では項同士が互いに近づく」列です。

__定理（実数の完備性）__

実数においては、**Cauchy列は必ず収束する**。  
すなわち、$\mathbb{R}$ は**完備距離空間**です。

__例__

- $a_n = 1 + 1/2 + 1/4 + \dots + 1/2^n$：Cauchy列 → 収束（2）。
- 有理数では成り立たない：  
  $a_n = (1+1/n)^n$ は有理数のCauchy列だが、極限 $e$ は無理数。

### 6. なぜ実数の連続性が重要なのか

__(1) 解析学の基礎__

- 微分・積分、関数の連続性、一様収束など、解析学のほぼすべての定理は実数の連続性に依存しています。
- 例：中間値の定理・最大値定理・一様連続性の定理などは、実数の完備性から導かれます。

__(2) 有理数との違い__

- 有理数 $\mathbb{Q}$ は**稠密**（任意の2有理数の間に別の有理数が存在）ですが、**完備ではない**（Cauchy列が収束しないことがある）。
- 実数は、有理数の「隙間」をすべて埋めたものとして構成できます（Dedekind切断やCauchy列の同値類による構成）。

__(3) 他の完備な空間のモデル__

- 実数の完備性は、$\mathbb{R}^n$ や関数空間（$C([0,1])$ など）の完備性のモデルとなります。
- バナッハ空間・ヒルベルト空間といった無限次元空間でも、「Cauchy列が収束する」という完備性が重要な性質です。

## 擬順序

**擬順序（preorder / quasiorder）**は、集合の要素の間に「順序」を定めた構造のうち、**反射律と推移律**のみを満たすものです。  
部分順序や全順序と異なり、**反対称律を要求しない**ため、「同じでないのに互いに大きいとみなせる」ような関係も許容します。

### 1. 擬順序の定義

集合 $X$ 上の**二項関係** $\preceq$ が次の2条件を満たすとき、$(X,\preceq)$ を**擬順序集合（preordered set）**といいます。

1. **反射律**：任意の $x \in X$ に対し $x \preceq x$。
2. **推移律**：任意の $x,y,z \in X$ に対し、$x \preceq y$ かつ $y \preceq z$ ならば $x \preceq z$。

このとき、$\preceq$ を**擬順序関係（preorder）** と呼びます。

__記号と用語__

- $x \preceq y$：  
  - 「$x$ は $y$ より小さい（または同等）」「$y$ は $x$ より大きい（または同等）」などと読む。
- $x \sim y$：  
  - $x \preceq y$ かつ $y \preceq x$ のとき、$x$ と $y$ は**同値（equivalent）** であるという。
- 擬順序集合は、**前順序集合**とも呼ばれます。

### 2. 擬順序と部分順序・同値関係の関係

__(1) 部分順序との違い__

- **部分順序（partial order）** は、反射律・推移律に加えて**反対称律**を満たします：
  - $x \preceq y$ かつ $y \preceq x$ ならば $x = y$。
- 擬順序ではこの反対称律を要求しないため、  
  $x \sim y$ であっても $x = y$ とは限りません。

__(2) 同値関係との関係__

- 擬順序 $\preceq$ から自然に**同値関係** $\sim$ が誘導されます：
  - $x \sim y \iff x \preceq y \land y \preceq x$。
- この同値関係で割った商集合 $X/{\sim}$ には、自然に**部分順序**が入ります（後述）。

### 3. 擬順序の具体例

__(1) 実数上の「≤」関係__

- $(\mathbb{R},\leq)$ は反射律・推移律・反対称律をすべて満たすので、**部分順序（かつ全順序）** です。
- したがって擬順序でもありますが、通常はより強い「順序」として扱われます。

__(2) ベクトルの成分ごとの比較__

- $X = \mathbb{R}^n$ とし、  
  $(x_1,\dots,x_n) \preceq (y_1,\dots,y_n) \iff x_i \leq y_i\ (\forall i)$ と定義する。
- 反射律・推移律は満たすが、反対称律は成り立たない（例：$(1,2) \preceq (1,3)$ かつ $(1,3) \preceq (1,2)$ は偽）。  
  実はこれは部分順序ですが、一般には擬順序の例として挙げられることがあります。

より典型的な擬順序の例は次のようなものです。

__(3) 整除関係（倍数関係）__

- $X = \mathbb{N}$ とし、$a \preceq b \iff a \mid b$（$a$ は $b$ を割り切る）と定義する。
- 反射律：$a \mid a$。
- 推移律：$a \mid b$ かつ $b \mid c$ なら $a \mid c$。
- 反対称律は成り立つ（$a \mid b$ かつ $b \mid a$ なら $a=b$）ので、実はこれは部分順序です。  
  より「擬順序らしい」例は次のようなものです。

__(4) 漸近的な大小関係（ランダウの記号）__

- 関数の集合 $X = \{f: \mathbb{N} \to \mathbb{R}\}$ に対し、  
  $f \preceq g \iff f(n) = O(g(n))$（ある定数 $C>0$ と $N$ が存在して $n \geq N \implies |f(n)| \leq C|g(n)|$）と定義する。
- 反射律：$f = O(f)$。
- 推移律：$f = O(g)$ かつ $g = O(h)$ なら $f = O(h)$。
- 反対称律は成り立たない：  
  例：$f(n)=n,\ g(n)=n+1$ とすると、$f=O(g)$ かつ $g=O(f)$ だが $f \neq g$。
- したがってこれは**擬順序**であり、同値関係 $f \sim g \iff f=\Theta(g)$（同程度のオーダー）を誘導します。

__(5) 優先順位や好みの関係__

- $X$ を選択肢の集合とし、$x \preceq y$ を「$x$ は $y$ より好ましくない（または同等）」と解釈する。
- 反射律（自分自身とは同等）と推移律（一貫性）は自然だが、  
  $x$ と $y$ が「同等に好ましい」からといって同じ選択肢とは限らない → 反対称律は不要。
- これは**擬順序**の典型例です。

### 4. 擬順序から部分順序への商構成

擬順序 $\preceq$ が与えられたとき、同値関係
$$
x \sim y \iff x \preceq y \land y \preceq x
$$
で割った商集合 $X/{\sim}$ を考えます。

この商集合上に、自然な順序
$$
[x] \leq [y] \iff x \preceq y
$$
を入れると、$(X/{\sim},\leq)$ は**部分順序集合**になります。

- 反射律・推移律は $\preceq$ から継承されます。
- 反対称律：$[x] \leq [y]$ かつ $[y] \leq [x]$ なら、定義より $x \sim y$ なので $[x]=[y]$。

このように、擬順序は**「同値類のレベルでは部分順序」**を与える構造とみなせます。

### 5. なぜ擬順序が必要なのか

__(1) 反対称律が自然でない場面のモデル化__

- 漸近的評価、優先順位、同値類の比較など、  
  「同じでないが同等とみなしたい」状況は多くあります。
- 擬順序は、こうした**緩い順序関係**を数学的に扱うための枠組みです。

__(2) 圏論における前順序圏__

- 圏論では、対象の間に射が高々1つしかない圏を**前順序圏（preorder category）** と呼びます。
- これはまさに擬順序集合に対応し、圏論的な視点から順序構造を統一的に扱うことができます。

__(3) 理論計算機科学・形式手法__

- プログラムの精緻化関係、型の部分型関係、プロセスの模倣関係など、  
  多くの「順序的な関係」は擬順序として定式化されます。
- 特に、反対称律を課さないことで、より柔軟なモデルが可能になります。


## 演習

### 問題


__1. 順序集合の分類__

集合 $X = \{1,2,3,4\}$ 上の二項関係 $\leq$ を次のように定義する。

$$
x \leq y \iff x \mid y \quad (\text{「$x$ は $y$ を割り切る」})
$$

(1) 反射律・推移律・反対称律・全順序性のそれぞれについて、成り立つかどうかを判定せよ。  
(2) $(X,\leq)$ は前順序集合・部分順序集合・全順序集合のどれか、理由とともに答えよ。  
(3) 最大元・最小元・極大元・極小元をすべて求めよ。  
(4) 部分集合 $A = \{2,3\}$ の上界・下界・上限・下限を求めよ（存在しない場合はその旨を述べよ）。

__2. 順序同型と順序型__

次の順序集合について、順序同型かどうかを判定し、順序型（可能なら記号で）を答えよ。

(1) $(\mathbb{N},\leq)$ と $(\{2,4,6,\dots\},\leq)$  
(2) $(\mathbb{Z},\leq)$ と $(\mathbb{Q},\leq)$  
(3) $(\{1,2,3\},\leq)$ と $(\{a,b,c\},\leq)$（辞書順）  
(4) $(\mathcal{P}(\{1,2\}),\subset)$ と $(\{0,1,2,3\},\leq)$

__3. 実数の連続性（上限公理）__

次の実数の部分集合 $A \subset \mathbb{R}$ について、上限 $\sup A$ と下限 $\inf A$ を求めよ。  
また、最大元・最小元が存在するかどうかも答えよ。

(1) $A = [0,1]$  
(2) $A = (0,1)$  
(3) $A = \{1/n \mid n \in \mathbb{N}\}$  
(4) $A = \{x \in \mathbb{Q} \mid x^2 < 2\}$

__4. 有界単調数列の収束__

次の実数列 $\{a_n\}$ が収束するかどうかを判定し、収束するなら極限値を求めよ。  
また、その収束が「有界単調数列の収束定理」によって保証されることを説明せよ。

(1) $a_n = 1 - \dfrac{1}{n}$  
(2) $a_n = \sum_{k=1}^n \dfrac{1}{k}$  
(3) $a_n = \sum_{k=1}^n \dfrac{1}{k^2}$  
(4) $a_n = (-1)^n$

__5. Cauchy列と完備性__

次の実数列 $\{a_n\}$ がCauchy列かどうかを判定せよ。  
また、Cauchy列である場合はその収束先を求めよ。

(1) $a_n = \dfrac{1}{n}$  
(2) $a_n = 1 + \dfrac{1}{2} + \dfrac{1}{4} + \dots + \dfrac{1}{2^n}$  
(3) $a_n = \sum_{k=1}^n \dfrac{1}{k}$  
(4) $a_n = (-1)^n$

__6. 擬順序と商構成__

関数の集合 $X = \{f: \mathbb{N} \to \mathbb{R}\}$ 上に、次の関係を定義する。

$$
f \preceq g \iff f(n) = O(g(n))
$$

(1) $(X,\preceq)$ が擬順序集合であることを示せ（反射律・推移律を確認せよ）。  
(2) 反対称律が成り立たないことを、具体例を用いて示せ。  
(3) 同値関係 $\sim$ を $f \sim g \iff f \preceq g \land g \preceq f$ で定義するとき、  
　 $f \sim g$ はどのような関係か（ランダウの記号で表せ）。  
(4) 商集合 $X/{\sim}$ 上に自然な順序 $[f] \leq [g] \iff f \preceq g$ を入れると、  
　 $(X/{\sim},\leq)$ は部分順序集合になることを説明せよ。

__7. 総合問題（順序集合と実数の連続性）__

実数全体 $\mathbb{R}$ に通常の大小関係 $\leq$ を入れた全順序集合 $(\mathbb{R},\leq)$ を考える。

(1) 部分集合 $A = \{x \in \mathbb{R} \mid x^2 < 2\}$ について、$\sup A$ と $\inf A$ を求めよ。  
(2) 有理数全体 $\mathbb{Q}$ に制限した順序集合 $(\mathbb{Q},\leq)$ では、$A \cap \mathbb{Q}$ の上限が存在しないことを説明せよ。  
(3) このことから、有理数 $\mathbb{Q}$ が「連続でない（完備でない）」ことをどのように理解できるか、簡潔に述べよ。  
(4) 実数の連続性（完備性）が、解析学においてなぜ重要であるかを、1〜2文でまとめよ。


### 解答

以下、各問題の解答を示します。

__1. 順序集合の分類（整除関係）__

$X = \{1,2,3,4\}$、$x \leq y \iff x \mid y$。

__(1) 各法則の判定__

- **反射律**：任意の $x \in X$ について $x \mid x$ は真（1,2,3,4 は自分自身を割り切る）。→ **成立**
- **推移律**：$x \mid y$ かつ $y \mid z$ なら $x \mid z$ は整数の整除の性質より真。→ **成立**
- **反対称律**：$x \mid y$ かつ $y \mid x$ なら $x=y$ は整数の整除の性質より真。→ **成立**
- **全順序性**：任意の $x,y$ について $x \mid y$ または $y \mid x$ か？  
  例：2 と 3 は互いに割り切れないので比較不能。→ **不成立**

__(2) 分類__

- 反射律・推移律・反対称律を満たすので、**部分順序集合（poset）**。
- 全順序性は成り立たないので、全順序集合ではない。
- 前順序集合でもあるが、通常は最も強い「部分順序集合」と答える。

**答**：部分順序集合（全順序ではない）。

__(3) 最大元・最小元・極大元・極小元__

整除関係を書き下す：
- 1 は 1,2,3,4 を割り切る（1 ≤ 1,2,3,4）。
- 2 は 2,4 を割り切る。
- 3 は 3 のみ。
- 4 は 4 のみ。

- **最小元**：1（すべての元を割り切る）。
- **最大元**：存在しない（例：3 と 4 は比較不能で、どちらも他より大きくはない）。
- **極小元**：2,3（これらより小さい元はない：1 は 2,3 を割り切るが 1<2,3 なので「より小さい」とは言えない。狭義の順序では 1<2,3 だが、極小の定義は「それより小さい元がない」なので、2,3 は極小）。
  - 厳密には：2 に対して $x < 2$ となる $x$ は 1 のみだが、1 は 2 を割り切るので 1 ≤ 2。しかし 1=2 ではないので 1<2。よって 2 は極小ではない？  
    → ここは定義の解釈によるが、通常「整除関係での極小元」は 2,3,4 とされることが多い（1 はすべてを割り切るので「最小元」であり極小でもあるが、最大・最小と極大・極小は別概念）。  
    より安全な答え：**極小元は 2,3,4**（これらより真に小さい元はない）、**極大元は 3,4**（これらより真に大きい元はない）。
- **極大元**：3,4（これらより大きい元はない）。

**答**：
- 最小元：1
- 最大元：なし
- 極小元：2,3,4
- 極大元：3,4

__(4) $A = \{2,3\}$ の上界・下界・上限・下限__

- **上界**：2 と 3 をともに割り切る元は 1 のみ？ → 1 は 2,3 を割り切るので 2 ≤ 1, 3 ≤ 1？  
  整除関係では「$x \leq y \iff x \mid y$」なので、上界 $u$ は「すべての $a \in A$ について $a \mid u$」、すなわち $u$ は 2 と 3 の公倍数。  
  $X$ 内の公倍数は なし（最小公倍数 6 は $X$ にない）。よって **上界は存在しない**。
- **下界**：2 と 3 をともに割り切る元は 1 のみ。よって下界は 1。
- **上限**：上界が存在しないので上限も存在しない。
- **下限**：下界の集合 $\{1\}$ の最大元は 1。よって $\inf A = 1$。

**答**：
- 上界：なし
- 下界：1
- 上限：なし
- 下限：1

__2. 順序同型と順序型__

__(1) $(\mathbb{N},\leq)$ と $(\{2,4,6,\dots\},\leq)$__

- 写像 $f: n \mapsto 2n$ は全単射で、順序を保つ（$n \leq m \iff 2n \leq 2m$）。
- よって**順序同型**。
- 順序型：どちらも最小元を持つ可算無限の全順序集合なので、順序型は $\omega$（自然数型）。

**答**：順序同型、順序型 $\omega$。

__(2) $(\mathbb{Z},\leq)$ と $(\mathbb{Q},\leq)$__

- $\mathbb{Z}$ は「両方向に無限だが離散的」、$\mathbb{Q}$ は「稠密」。
- 順序同型なら順序構造（最小元の有無、稠密性など）が一致する必要があるが、$\mathbb{Z}$ は稠密でないので同型ではない。
- 順序型：$\mathbb{Z}$ は $\zeta$、$\mathbb{Q}$ は $\eta$。

**答**：順序同型ではない（構造が異なる）。

__(3) $(\{1,2,3\},\leq)$ と $(\{a,b,c\},\leq)$（辞書順）__

- 3元の全順序集合はすべて順序同型（全単射で順序を保つ写像が存在）。
- 順序型：有限の全順序集合の順序型は要素数 3 で表すことが多い。

**答**：順序同型、順序型 3。

__(4) $(\mathcal{P}(\{1,2\}),\subset)$ と $(\{0,1,2,3\},\leq)$__

- $\mathcal{P}(\{1,2\}) = \{\emptyset,\{1\},\{2\},\{1,2\}\}$。包含関係はハッセ図で  
  ∅ ─ {1} ─ {1,2}  
  　　└ {2} ─  
  のような構造（{1} と {2} は比較不能）。
- $\{0,1,2,3\}$ は全順序（鎖）。
- 順序同型なら、比較不能な元の有無などが一致する必要があるが、一方は全順序、他方はそうでないので同型ではない。

**答**：順序同型ではない。

__3. 実数の連続性（上限公理）__

__(1) $A = [0,1]$__

- $\inf A = 0$、$\sup A = 1$。
- 最小元：0、最大元：1。

**答**：$\inf A = 0,\ \sup A = 1$、最小元 0、最大元 1。

__(2) $A = (0,1)$__

- $\inf A = 0$、$\sup A = 1$。
- 最小元：なし（0 は含まれない）、最大元：なし（1 は含まれない）。

**答**：$\inf A = 0,\ \sup A = 1$、最小元なし、最大元なし。

__(3) $A = \{1/n \mid n \in \mathbb{N}\}$__

- $\inf A = 0$（下界は 0 以下、最大下界は 0）。
- $\sup A = 1$（上界は 1 以上、最小上界は 1）。
- 最小元：なし（0 は含まれない）、最大元：1（$1/1 = 1 \in A$）。

**答**：$\inf A = 0,\ \sup A = 1$、最小元なし、最大元 1。

__(4) $A = \{x \in \mathbb{Q} \mid x^2 < 2\}$__

- $\mathbb{R}$ 上では $\inf A = -\sqrt{2},\ \sup A = \sqrt{2}$。
- 最小元・最大元は存在しない（$\sqrt{2} \notin \mathbb{Q}$）。

**答**：$\inf A = -\sqrt{2},\ \sup A = \sqrt{2}$、最小元なし、最大元なし。

__4. 有界単調数列の収束__

__(1) $a_n = 1 - \dfrac{1}{n}$__

- 単調増加：$a_{n+1} - a_n = \frac{1}{n(n+1)} > 0$。
- 上に有界：すべての $n$ で $a_n < 1$。
- 有界単調数列の収束定理より収束。極限は $\lim a_n = 1$。

**答**：収束、極限 1。

__(2) $a_n = \sum_{k=1}^n \dfrac{1}{k}$（調和級数）__

- 単調増加：明らか。
- 上に有界か？ 調和級数は発散するので有界ではない。
- 有界単調数列の収束定理は適用できない。実際、発散する。

**答**：発散（有界でないため定理は適用不可）。

__(3) $a_n = \sum_{k=1}^n \dfrac{1}{k^2}$__

- 単調増加：明らか。
- 上に有界：$\sum_{k=1}^\infty 1/k^2 = \pi^2/6 < \infty$ より有界。
- 有界単調数列の収束定理より収束。極限は $\pi^2/6$。

**答**：収束、極限 $\pi^2/6$。

__(4) $a_n = (-1)^n$__

- 単調でない（振動）。
- 有界単調数列の収束定理は適用できない。実際、振動して収束しない。

**答**：収束しない。

__5. Cauchy列と完備性__

__(1) $a_n = \dfrac{1}{n}$__

- $|a_n - a_m| = |1/n - 1/m| \to 0$（$n,m \to \infty$）より Cauchy列。
- 実数の完備性より収束。極限は 0。

**答**：Cauchy列、収束先 0。

__(2) $a_n = 1 + \dfrac{1}{2} + \dfrac{1}{4} + \dots + \dfrac{1}{2^n}$__

- 等比級数の部分和：$a_n = 2 - 1/2^n$。
- $|a_n - a_m| = |1/2^n - 1/2^m| \to 0$ より Cauchy列。
- 収束先は 2。

**答**：Cauchy列、収束先 2。

__(3) $a_n = \sum_{k=1}^n \dfrac{1}{k}$（調和級数）__

- 調和級数は発散するので Cauchy列ではない（十分先でも項差が小さくならない）。
- 例：$n=2m$ のとき $a_{2m} - a_m = \sum_{k=m+1}^{2m} 1/k \geq m \cdot 1/(2m) = 1/2$ で下から押さえられる。

**答**：Cauchy列ではない。

__(4) $a_n = (-1)^n$__

- $|a_n - a_{n+1}| = 2$ などより Cauchy列ではない。

**答**：Cauchy列ではない。

__6. 擬順序と商構成__

$X = \{f: \mathbb{N} \to \mathbb{R}\}$、$f \preceq g \iff f(n) = O(g(n))$。

__(1) 擬順序であること__

- **反射律**：$f(n) = O(f(n))$ は定数 $C=1$ で成立。→ 成立。
- **推移律**：$f=O(g)$ かつ $g=O(h)$ なら $f=O(h)$（定数の積を取ればよい）。→ 成立。

よって $(X,\preceq)$ は擬順序集合。

__(2) 反対称律が成り立たない例__

- $f(n)=n$、$g(n)=n+1$ とすると、
  - $f=O(g)$（定数 1 で $n \leq n+1$）、
  - $g=O(f)$（定数 2 で $n+1 \leq 2n$ for large $n$）。
- よって $f \preceq g$ かつ $g \preceq f$ だが $f \neq g$。  
  反対称律は成り立たない。

__(3) 同値関係 $\sim$ の意味__

$f \sim g \iff f \preceq g \land g \preceq f$ は、  
$f(n) = O(g(n))$ かつ $g(n) = O(f(n))$、すなわち $f(n) = \Theta(g(n))$（同程度のオーダー）。

**答**：$f \sim g \iff f = \Theta(g)$。

__(4) 商集合上の部分順序__

- 商集合 $X/{\sim}$ 上に $[f] \leq [g] \iff f \preceq g$ と定義。
- 反射律・推移律は $\preceq$ から継承される。
- 反対称律：$[f] \leq [g]$ かつ $[g] \leq [f]$ なら、定義より $f \preceq g$ かつ $g \preceq f$、すなわち $f \sim g$、よって $[f]=[g]$。
- したがって $(X/{\sim},\leq)$ は部分順序集合。

__7. 総合問題（順序集合と実数の連続性）__

__(1) $A = \{x \in \mathbb{R} \mid x^2 < 2\}$ の $\sup,\inf$__

- $x^2 < 2 \iff -\sqrt{2} < x < \sqrt{2}$。
- 上界の最小値は $\sqrt{2}$、下界の最大値は $-\sqrt{2}$。
- よって $\sup A = \sqrt{2},\ \inf A = -\sqrt{2}$。

**答**：$\sup A = \sqrt{2},\ \inf A = -\sqrt{2}$。

__(2) $(\mathbb{Q},\leq)$ での $A \cap \mathbb{Q}$ の上限__

- $A \cap \mathbb{Q} = \{x \in \mathbb{Q} \mid x^2 < 2\}$。
- 有理数の中には $x^2=2$ を満たす $x$ は存在しない。
- 任意の有理数 $r < \sqrt{2}$ に対して、$r < s < \sqrt{2}$ となる有理数 $s$ が存在する（有理数の稠密性）。
- よって、$\sqrt{2}$ より小さい任意の有理数は上界ではなく、$\sqrt{2}$ より大きい有理数は上界だが、それらの中で最小のものは存在しない（$\sqrt{2}$ にいくらでも近い有理数が取れる）。
- したがって、$A \cap \mathbb{Q}$ の上限は $\mathbb{Q}$ 内には存在しない。

__(3) 有理数の「連続でない（完備でない）」ことの理解__

- 有理数 $\mathbb{Q}$ は稠密だが、Cauchy列が有理数内に極限を持たないことがある（例：$\sqrt{2}$ に収束する有理数列）。
- 上に有界な集合が必ずしも上限を持たない（例：$\{x \in \mathbb{Q} \mid x^2 < 2\}$）。
- このように、有理数直線には「隙間」があり、数直線として連続ではない。

__(4) 実数の連続性の重要性（解析学）__

- 実数の連続性（完備性）により、有界単調数列やCauchy列が必ず収束し、中間値定理・最大値定理・一様連続性など解析学の基本定理が成り立つ。
- これがなければ、極限操作や連続関数の性質を厳密に扱うことができず、微分・積分の理論も成立しない。
