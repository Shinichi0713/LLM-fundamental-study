# 基数と濃度

## 基数と濃度

集合論における「基数（cardinal number）」と「濃度（cardinality）」は、集合の「大きさ」を表す概念です。多くの文脈ではほぼ同じ意味で使われますが、厳密には次のように整理できます。

### 1. 濃度（cardinality）の定義

__1.1 集合の「大きさ」の直観__
- 有限集合なら「要素の個数」で大きさを表せます。
- 無限集合でも、たとえば自然数全体の集合と偶数全体の集合は「同じ大きさ」とみなしたい（全単射が存在する）という直観があります。
- この「大きさ」を一般化したのが濃度です。

__1.2 濃度の形式的な定義（公理的集合論）__

公理的集合論（ZFCなど）では、次のように定義されることが多いです。

- 集合 $A$ の濃度を $|A|$ と書きます。
- 集合 $A$ と $B$ について、「$A$ と $B$ の間に全単射が存在する」とき、$|A| = |B|$ と定義します。
- 濃度は「全単射が存在する」という同値関係による同値類として定義されます。

より厳密には、選択公理（AC）を仮定すると、任意の集合 $A$ に対して  
「$A$ と全単射で結びつく最小の順序数」を $A$ の**基数（cardinal number）** と定義し、それを $|A|$ と書きます。

### 2. 基数（cardinal number）の定義

__2.1 基数とは__
- 基数とは、「濃度を表す順序数」のことです。
- 順序数は「整列順序の型」を表す数ですが、その中で「自分より小さい順序数との間に全単射が存在しない」ものを**基数**と呼びます。
- 例：
  - 有限順序数 $0,1,2,\dots$ はすべて基数です。
  - 最小の無限順序数 $\omega$ は基数であり、$\aleph_0$ と書きます（可算無限集合の濃度）。

__2.2 基数の記号__
- 有限基数：$0,1,2,\dots$
- 可算無限基数：$\aleph_0$
- その次の無限基数：$\aleph_1, \aleph_2, \dots$（アレフ数）

### 3. 濃度と基数の関係

- **濃度**：集合の「大きさ」そのもの（同値類としての概念）。
- **基数**：その大きさを表す「代表元」として選ばれた順序数。
- 多くの場合、同じ意味で使われますが、厳密には
  - 「濃度」は同値類
  - 「基数」はその同値類の代表として選ばれた順序数
  という関係です。

### 4. 濃度の比較（大小関係）

__4.1 定義__
- $|A| \le |B|$ とは、「$A$ から $B$ への単射が存在する」ことと定義します。
- $|A| < |B|$ とは、「$|A| \le |B|$ かつ $|A| \neq |B|$」と定義します。

__4.2 カントールの定理__
任意の集合 $A$ について、
$$
|A| < |\mathcal{P}(A)|
$$
が成り立ちます（$\mathcal{P}(A)$ は $A$ の冪集合）。  
特に、無限集合には「より大きい無限集合」が必ず存在します。

### 5. 主な濃度の例

__5.1 有限濃度__
- $|A| = n$（自然数） ⇔ $A$ は $n$ 個の要素を持つ有限集合。

__5.2 可算無限濃度__
- $|\mathbb{N}| = |\mathbb{Z}| = |\mathbb{Q}| = \aleph_0$
- 自然数と全単射がつくれる集合を「可算無限集合」といいます。

__5.3 連続体濃度__
- $|\mathbb{R}| = |(0,1)| = |\mathcal{P}(\mathbb{N})| = 2^{\aleph_0}$
- これを連続体濃度と呼び、$\mathfrak{c}$ と書くこともあります。

### 6. 濃度の演算

基数 $\kappa, \lambda$ に対して、次のように定義します（無限基数の場合も含む）。

- **和**：$\kappa + \lambda = |A \cup B|$（ただし $A \cap B = \varnothing$ で $|A|=\kappa, |B|=\lambda$）
- **積**：$\kappa \cdot \lambda = |A \times B|$
- **冪**：$\kappa^\lambda = |\{f \mid f \colon B \to A\}|$（ただし $|A|=\kappa, |B|=\lambda$）

__6.1 無限基数の演算の性質__
- $\kappa$ が無限基数のとき：
  - $\kappa + \lambda = \kappa \cdot \lambda = \max(\kappa, \lambda)$
  - 特に $\aleph_0 + n = \aleph_0$、$\aleph_0 + \aleph_0 = \aleph_0$ など。
- 冪演算は一般に異なる値を取り得ます（例：$2^{\aleph_0} > \aleph_0$）。

### 7. 連続体仮説（CH）と一般連続体仮説（GCH）

- **連続体仮説**：$2^{\aleph_0} = \aleph_1$ かどうかという問題。
- **一般連続体仮説**：任意の基数 $\kappa$ について $2^{\kappa} = \kappa^+$（$\kappa$ の次の基数）かどうかという問題。
- これらは ZFC から独立であることが知られています（証明も反証もできない）。


## 基数の例

基数（cardinal number）の例を、有限・可算無限・非可算無限の順に挙げます。

### 1. 有限基数の例

有限集合の要素の個数として現れる基数です。

- $| \varnothing | = 0$
- $| \{a\} | = 1$
- $| \{1,2,3\} | = 3$
- 一般に、$n$ 個の要素からなる有限集合の濃度は $n$（自然数）です。

### 2. 可算無限基数の例

自然数全体の集合 $\mathbb{N}$ と全単射が存在する集合の濃度を「可算無限濃度」といい、その基数を $\aleph_0$ と書きます。

- $|\mathbb{N}| = \aleph_0$
- $|\mathbb{Z}| = \aleph_0$（整数全体）
- $|\mathbb{Q}| = \aleph_0$（有理数全体）
- $|\mathbb{N} \times \mathbb{N}| = \aleph_0$（自然数の組全体）
- 偶数全体、素数全体、整数係数の多項式全体なども、いずれも $\aleph_0$

これらはすべて「自然数と 1 対 1 に対応づけられる」という意味で同じ大きさです。

### 3. 非可算無限基数の例

自然数と全単射が作れない無限集合の濃度です。

__3.1 連続体濃度（$2^{\aleph_0}$）__

- $|\mathbb{R}| = 2^{\aleph_0}$（実数全体）
- $|(0,1)| = 2^{\aleph_0}$（開区間）
- $|\mathcal{P}(\mathbb{N})| = 2^{\aleph_0}$（自然数の部分集合全体）
- 複素数全体 $\mathbb{C}$ も同じ濃度です。

__3.2 より大きな無限基数__

- $|\mathcal{P}(\mathbb{R})| = 2^{2^{\aleph_0}}$（実数の部分集合全体）
- 一般に、集合 $A$ に対して $|\mathcal{P}(A)| = 2^{|A|}$ であり、カントールの定理により
  $$
  |A| < 2^{|A|}
  $$
  が成り立ちます。

したがって、無限基数の列
$$
\aleph_0,\ 2^{\aleph_0},\ 2^{2^{\aleph_0}},\ 2^{2^{2^{\aleph_0}}},\ \dots
$$
はどんどん大きくなっていきます。

### 4. アレフ数（$\aleph_\alpha$）としての基数

順序数 $\alpha$ に対して、$\aleph_\alpha$ という無限基数が定義されます。

- $\aleph_0$：可算無限基数
- $\aleph_1$：$\aleph_0$ の「次の」基数（最小の非可算基数）
- $\aleph_2$：その次の基数
- 一般に、$\aleph_\alpha$ は「それより小さい基数との間に全単射が存在しない」最小の順序数として定義されます。

連続体仮説（CH）は
$$
2^{\aleph_0} = \aleph_1
$$
が成り立つかどうかという問題ですが、これは ZFC から独立しています。

## 特異基数と正則基数

以下、「特異基数」と「正則基数」について、数学的な定義と特徴を説明します。

### 1. 正則基数（regular cardinal）

__1.1 定義__

基数 $\kappa$ が**正則（regular）** であるとは、次の条件を満たすことをいいます。

> 任意の集合族 $\{A_\alpha\}_{\alpha<\lambda}$ について、
> - 各 $A_\alpha$ の濃度が $\kappa$ 未満（$|A_\alpha| < \kappa$）
> - 添字集合の濃度も $\kappa$ 未満（$\lambda < \kappa$） 
>ならば、その和集合の濃度も $\kappa$ 未満である：
$$
\left|\bigcup_{\alpha<\lambda} A_\alpha\right| < \kappa.
$$
>

言い換えると：

- 「小さな集合（濃度 $<\kappa$）を、それより小さい個数（$<\kappa$ 個）だけ集めても、全体の大きさは $\kappa$ に達しない」

という性質です。

__1.2 同値な特徴づけ（共終数を用いた定義）__

基数 $\kappa$ の**共終数（cofinality）** を $\mathrm{cf}(\kappa)$ と書きます。これは

> $\kappa$ に共終な（cofinal）増加列の最小の長さ

として定義されます。

このとき、次が成り立ちます。

- $\kappa$ が正則 ⇔ $\mathrm{cf}(\kappa) = \kappa$
- $\kappa$ が特異（後述） ⇔ $\mathrm{cf}(\kappa) < \kappa$

__1.3 正則基数の例__

- すべての**有限基数**は正則です。
- 最小の無限基数 $\aleph_0$ は正則です。
- 後続基数（successor cardinal）は常に正則です：
  - $\aleph_1, \aleph_2, \dots, \aleph_{\alpha+1}, \dots$ はすべて正則。
- 到達不能基数（inaccessible cardinal）も正則です。

__1.4 正則基数の性質__

- 正則基数は「分解しにくい」基数です。
- 順序数として見ると、「自分より小さい順序数の列で、極限として自分自身に到達できない」という性質を持ちます。
- 正則基数は、多くの組合せ論的・モデル論的性質（木性質、分割性質など）を持つことが知られています。

### 2. 特異基数（singular cardinal）

__2.1 定義__

基数 $\kappa$ が**特異（singular）** であるとは、**正則でない**こと、すなわち次が成り立つことをいいます。

> ある集合族 $\{A_\alpha\}_{\alpha<\lambda}$ が存在して、
> - 各 $A_\alpha$ の濃度が $\kappa$ 未満（$|A_\alpha| < \kappa$）
> - 添字集合の濃度も $\kappa$ 未満（$\lambda < \kappa$）
> 
> であるにもかかわらず、和集合の濃度が $\kappa$ に等しい：
$$
\left|\bigcup_{\alpha<\lambda} A_\alpha\right| = \kappa.
$$
>

共終数を用いると：

- $\kappa$ が特異 ⇔ $\mathrm{cf}(\kappa) < \kappa$

__2.2 特異基数の例__

- $\aleph_\omega = \sup\{\aleph_n \mid n<\omega\}$ は特異基数です。
  - 各 $\aleph_n$ は $\aleph_\omega$ より小さい基数。
  - 可算無限個（$\omega$ 個）の和集合として $\aleph_\omega$ に到達する。
  - よって $\mathrm{cf}(\aleph_\omega) = \omega < \aleph_\omega$。
- 一般に、極限順序数 $\lambda$ に対して $\aleph_\lambda$ が特異になることがあります。
  - 例：$\aleph_{\omega+\omega},\ \aleph_{\omega_1}$ など。

__2.3 特異基数の性質__

- 特異基数は「小さな基数の集まりとして表現できる」基数です。
- 特異基数仮説（Singular Cardinals Hypothesis, SCH）など、特異基数に関する組合せ論的性質は集合論の重要な研究対象です。
- 特異基数は、正則基数に比べて「分解可能」であり、その振る舞いはしばしばより複雑です。

### 3. 正則基数と特異基数の関係

__3.1 基本的な関係__

- 任意の無限基数 $\kappa$ について、
  - $\mathrm{cf}(\kappa)$ は常に正則基数です。
  - 特に $\mathrm{cf}(\kappa)$ は $\kappa$ 以下の正則基数です。
- したがって、
  - $\kappa$ が正則 ⇔ $\mathrm{cf}(\kappa) = \kappa$
  - $\kappa$ が特異 ⇔ $\mathrm{cf}(\kappa) < \kappa$

__3.2 後続基数と極限基数__

- **後続基数**：ある基数 $\lambda$ に対して $\kappa = \lambda^+$ と書ける基数。
  - 例：$\aleph_1 = \aleph_0^+,\ \aleph_{\alpha+1} = \aleph_\alpha^+$
  - 後続基数は常に正則です。
- **極限基数**：後続基数でない無限基数。
  - 例：$\aleph_\omega,\ \aleph_{\omega_1}$ など。
  - 極限基数は正則であることも特異であることもあります。

__3.3 正則・特異の判定の直感的イメージ__

- 正則基数：
  - 「階段を一段ずつ上がっていっても、有限ステップや可算ステップでは頂上に達しない」
  - 例：$\aleph_1$ は可算個の可算集合の和では到達できない。
- 特異基数：
  - 「より小さい基数を、それより小さい個数だけ集めると、自分自身に到達してしまう」
  - 例：$\aleph_\omega$ は可算個の $\aleph_n$ の和として得られる。

## 基数の演算

以下、基数の演算について説明します。

### 1. 基数の演算の定義

基数 $\kappa, \lambda$ に対し、次の3つの演算が定義されます。

__1.1 和（加法）__

**定義**：  
$A, B$ を互いに素な集合とし、$|A| = \kappa,\ |B| = \lambda$ とする。このとき、

$$
\kappa + \lambda = |A \cup B|
$$

と定義する。

- 互いに素でない場合も、$A' = A \times \{0\},\ B' = B \times \{1\}$ のようにして互いに素な集合に取り替えればよい。
- 有限基数の場合は通常の自然数の和と一致します。

__1.2 積（乗法）__

**定義**：  
$|A| = \kappa,\ |B| = \lambda$ とする。このとき、

$$
\kappa \cdot \lambda = |A \times B|
$$

と定義する。

- 直積集合の要素数として定義されます。
- 有限基数の場合は通常の自然数の積と一致します。

__1.3 冪（べき乗）__

**定義**：  
$|A| = \kappa,\ |B| = \lambda$ とする。このとき、

$$
\kappa^\lambda = \left|\{f \mid f \colon B \to A\}\right|
$$

と定義する。

- すなわち、$B$ から $A$ への写像全体の集合の濃度です。
- 有限基数の場合、$n^m$ は通常の自然数のべき乗と一致します。

### 2. 有限基数の演算

有限基数（自然数）については、通常の算術と一致します。

- $m + n$：要素数 $m$ の集合と $n$ の集合の和集合の要素数。
- $m \cdot n$：直積集合の要素数。
- $m^n$：$n$ 元集合から $m$ 元集合への写像の個数。

### 3. 無限基数の演算の性質

__3.1 和と積の簡約法則__

$\kappa$ を無限基数とすると、任意の基数 $\lambda$ に対し、次が成り立ちます。

1. **和**：
   $$
   \kappa + \lambda = \max(\kappa, \lambda)
   $$

2. **積**：
   $$
   \kappa \cdot \lambda = \max(\kappa, \lambda)
   $$

特に：

- $\aleph_0 + n = \aleph_0$（$n$ は有限）
- $\aleph_0 + \aleph_0 = \aleph_0$
- $\aleph_0 \cdot \aleph_0 = \aleph_0$
- $\aleph_1 + \aleph_0 = \aleph_1$
- $\aleph_1 \cdot \aleph_0 = \aleph_1$

**直感的な理由**：  
無限集合は「有限個や可算個の要素を加えたり、有限個や可算個のコピーを取ったりしても、濃度は変わらない」からです。

__3.2 冪演算の性質__

冪演算は一般に和・積とは異なり、単純な $\max$ にはなりません。

- $2^{\aleph_0} > \aleph_0$（カントールの定理）
- 一般に、$2^\kappa > \kappa$（カントールの定理）

また、次の指数法則が成り立ちます（基数としての演算）。

1. **積の冪**：
   $$
   \kappa^{\lambda+\mu} = \kappa^\lambda \cdot \kappa^\mu
   $$

2. **冪の冪**：
   $$
   (\kappa^\lambda)^\mu = \kappa^{\lambda \cdot \mu}
   $$

3. **積の冪（一般には成り立たない）**：
   $$
   (\kappa \cdot \lambda)^\mu = \kappa^\mu \cdot \lambda^\mu
   $$
   は一般には成り立ちません（有限のときは成り立つが、無限では反例あり）。

### 4. 具体例

__4.1 可算無限基数 $\aleph_0$ の演算__

- $\aleph_0 + n = \aleph_0$
- $\aleph_0 + \aleph_0 = \aleph_0$
- $\aleph_0 \cdot n = \aleph_0$（$n$ 有限、$n>0$）
- $\aleph_0 \cdot \aleph_0 = \aleph_0$
- $n^{\aleph_0} = 2^{\aleph_0}$（$n \ge 2$ のとき）
- $\aleph_0^{\aleph_0} = 2^{\aleph_0}$

__4.2 連続体濃度 $2^{\aleph_0}$ の演算__

- $2^{\aleph_0} + \aleph_0 = 2^{\aleph_0}$
- $2^{\aleph_0} \cdot \aleph_0 = 2^{\aleph_0}$
- $2^{\aleph_0} \cdot 2^{\aleph_0} = 2^{\aleph_0}$
- $(2^{\aleph_0})^{\aleph_0} = 2^{\aleph_0 \cdot \aleph_0} = 2^{\aleph_0}$

### 5. 基数演算と順序数演算の違い

- **順序数演算**：整列順序の「型」を表す演算（和は連結、積は辞書式順序など）。
- **基数演算**：集合の「大きさ（濃度）」のみを考える演算。

例：

- 順序数の和：$1+\omega = \omega \neq \omega+1$
- 基数の和：$1 + \aleph_0 = \aleph_0 = \aleph_0 + 1$

基数演算では順序の情報を忘れるため、非可換性などは現れません（和・積は可換）。

## 連続体仮説

以下、「連続体仮説（Continuum Hypothesis, CH）」について、数学的な定義と特徴を説明します。

### 1. 連続体仮説の定義

__1.1 連続体濃度__

- 自然数全体の集合 $\mathbb{N}$ の濃度を**可算無限濃度**といい、その基数を $\aleph_0$ と書きます。
- 実数全体の集合 $\mathbb{R}$ の濃度を**連続体濃度（continuum）** といい、$|\mathbb{R}| = 2^{\aleph_0}$ と書きます。

__1.2 連続体仮説（CH）の主張__

**連続体仮説**とは、次の命題のことです。

> 連続体濃度 $2^{\aleph_0}$ は、可算無限濃度 $\aleph_0$ の**直後の基数**である：
$$
2^{\aleph_0} = \aleph_1.
$$


ここで $\aleph_1$ は、$\aleph_0$ より大きい最小の基数（最小の非可算基数）を表します。

### 2. 一般連続体仮説（GCH）

**一般連続体仮説（Generalized Continuum Hypothesis, GCH）** は、CH を任意の基数に拡張したものです。

> 任意の無限基数 $\kappa$ について、
$$
2^\kappa = \kappa^+
$$
> が成り立つ。

ここで $\kappa^+$ は $\kappa$ の**次の基数（successor cardinal）** を表します。

- CH は GCH の $\kappa = \aleph_0$ の場合に相当します。

### 3. 連続体仮説の特徴

__3.1 歴史的背景__

- カントールは、実数全体 $\mathbb{R}$ が自然数 $\mathbb{N}$ より「真に大きい」ことを示しました（対角線論法）。
- さらに彼は、「$\mathbb{R}$ の濃度は $\aleph_0$ の次の基数 $\aleph_1$ に等しいのではないか」と予想しました。これが連続体仮説です。
- 長い間、CH は真か偽かが未解決の問題でした。

__3.2 ZFC からの独立性__

- ゲーデル（1938）は、ZFC が無矛盾ならば **ZFC + CH** も無矛盾であることを示しました（構成可能宇宙 $L$ において CH が成り立つ）。
- コーエン（1963）は、強制法（forcing）を用いて、ZFC が無矛盾ならば **ZFC + ¬CH** も無矛盾であることを示しました。
- したがって、CH は **ZFC から独立**（証明も反証もできない）であることがわかります。

同様に、GCH も ZFC から独立であることが知られています。

__3.3 集合論における位置づけ__

- CH は ZFC の**追加公理**として採用することも、採用しないこともできます。
- 集合論の多くの結果は「ZFC + CH」や「ZFC + ¬CH」のどちらを仮定するかによって異なります。
- 実解析・記述集合論・測度論など、実数に関わる分野では、CH の仮定の有無が定理の成立に影響することがあります。

### 4. CH が成り立つ世界と成り立たない世界

__4.1 CH が成り立つ世界（$2^{\aleph_0} = \aleph_1$）__

- 実数全体 $\mathbb{R}$ の濃度は最小の非可算基数 $\aleph_1$ に等しい。
- 実数の部分集合の「サイズ」の種類が比較的少ない：
  - 可算集合
  - 連続体濃度の集合
- 実数上の測度・カテゴリーに関するいくつかの命題が単純化されることがあります。

__4.2 CH が成り立たない世界（$2^{\aleph_0} > \aleph_1$）__

- 実数全体 $\mathbb{R}$ の濃度は $\aleph_1$ より大きい（例：$2^{\aleph_0} = \aleph_2$ など）。
- 実数の部分集合の「中間的なサイズ」が存在しうる：
  - 濃度が $\aleph_0$ と $2^{\aleph_0}$ の間にある集合が存在しうる（ただし、これはさらに別の公理に依存）。
- 強制法を用いて、さまざまな $2^{\aleph_0}$ の値を実現するモデルを構成できます。

### 5. CH と関連する概念

__5.1 ベール空間・完全集合性質__

- 実数上の「ベールのカテゴリー定理」や「完全集合性質」などは、CH の仮定の有無によって性質が変わることがあります。
- 例えば、CH を仮定すると「非可算な実数部分集合で、完全集合を含まないもの」が存在しうるが、CH を否定する公理系ではそのような集合が存在しないこともあります。

__5.2 巨大基数との関係__

- 巨大基数公理を仮定すると、実数集合論の構造がより「整った」ものになり、CH の否定と整合的なモデルが構成しやすくなることが知られています。
- しかし、巨大基数公理そのものは CH を決定しません。

### 解答

以下、各問の解答です。

__問1（濃度と基数の定義）__

__(1) 濃度の定義__

集合 $A$ の**濃度** $|A|$ は、次の同値関係による同値類として定義されます。

- 集合 $A$ と $B$ の間に全単射が存在するとき、$|A| = |B|$ と定義する。
- この同値関係による同値類を「濃度」と呼ぶ。

__(2) 基数の定義（順序数を用いて）__

ZFC において、**基数（cardinal number）** は次のように定義されます。

- 順序数 $\alpha$ が**基数**であるとは、$\alpha$ より小さい任意の順序数 $\beta < \alpha$ に対し、$|\beta| \ne |\alpha|$ が成り立つことをいう。
- すなわち、$\alpha$ は「自分より小さい順序数との間に全単射が存在しない」最小の順序数である。

また、任意の集合 $A$ に対し、選択公理（整列可能定理）により $A$ を整列させ、その順序型として得られる順序数のうち、最小のものを $A$ の**基数**と定義し、それを $|A|$ と書きます。

__(3) 濃度と基数の関係__

- **濃度**：全単射が存在するという同値関係による**同値類**。
- **基数**：その同値類から「代表元」として選ばれた**順序数**。

多くの文脈では同じ記号 $|A|$ で表され、実質的に同じ意味で使われますが、厳密には
- 濃度は「大きさのクラス」
- 基数は「そのクラスの代表として選ばれた順序数」
という関係にあります。

__問2（濃度の比較）__

__(1) 濃度の大小関係の定義__

濃度の大小関係 $|A| \le |B|$ は、次のように定義します。

- $|A| \le |B|$ ⇔ $A$ から $B$ への単射が存在する。

また、$|A| < |B|$ は $|A| \le |B|$ かつ $|A| \ne |B|$ と定義します。

__(2) カントールの定理の証明__

**定理**：任意の集合 $A$ について $|A| < |\mathcal{P}(A)|$。

**証明**：

1. $|A| \le |\mathcal{P}(A)|$ を示す。  
   - 写像 $f: A \to \mathcal{P}(A)$ を $f(a) = \{a\}$ と定義すると、これは単射である。
   - よって $|A| \le |\mathcal{P}(A)|$。

2. $|A| \ne |\mathcal{P}(A)|$（すなわち全単射が存在しない）を示す。  
   - 背理法で、全単射 $g: A \to \mathcal{P}(A)$ が存在すると仮定する。
   - $B = \{a \in A \mid a \notin g(a)\}$ とおく。$B \subset A$ なので $B \in \mathcal{P}(A)$。
   - $g$ は全射なので、ある $b \in A$ が存在して $g(b) = B$。
   - このとき、
     - $b \in B$ とすると $b \in g(b) = B$ だが、$B$ の定義より $b \notin g(b)$ となり矛盾。
     - $b \notin B$ とすると $b \notin g(b)$ なので $B$ の定義より $b \in B$ となり矛盾。
   - いずれも矛盾するので、そのような全単射 $g$ は存在しない。

3. よって $|A| < |\mathcal{P}(A)|$。

__(3) 濃度の大小比較__

- $|\mathbb{N}| = |\mathbb{Z}| = |\mathbb{Q}| = \aleph_0$
- $|\mathbb{R}| = 2^{\aleph_0}$
- $|\mathcal{P}(\mathbb{N})| = 2^{\aleph_0}$

したがって、

- $|\mathbb{N}| = |\mathbb{Z}| = |\mathbb{Q}| < |\mathbb{R}| = |\mathcal{P}(\mathbb{N})|$

__問3（可算無限集合）__

__(1) 可算無限集合の定義__

集合 $A$ が**可算無限集合**であるとは、$A$ が自然数全体の集合 $\mathbb{N}$ と全単射で結びつくこと、すなわち $|A| = \aleph_0$ であることをいう。

__(2) 可算無限集合であることの証明__

**(a) 偶数全体 $\{2,4,6,\dots\}$**

- 写像 $f: \mathbb{N} \to \{2,4,6,\dots\}$ を $f(n) = 2(n+1)$ と定義すると、全単射。
- よって可算無限集合。

**(b) 整数全体 $\mathbb{Z}$**

- 写像 $f: \mathbb{N} \to \mathbb{Z}$ を
  $$
  f(n) = \begin{cases}
  -\frac{n}{2} & (n \text{が偶数}) \\
  \frac{n+1}{2} & (n \text{が奇数})
  \end{cases}
  $$
  と定義すると、全単射。
- よって可算無限集合。

**(c) 有理数全体 $\mathbb{Q}$**

- 有理数は既約分数 $p/q$（$p \in \mathbb{Z},\ q \in \mathbb{N}_{>0}$）として表せる。
- 写像 $(p,q) \mapsto p/q$ は $\mathbb{Z} \times \mathbb{N}_{>0}$ から $\mathbb{Q}$ への全射。
- $\mathbb{Z} \times \mathbb{N}_{>0}$ は可算無限集合（$\mathbb{Z}$ と $\mathbb{N}_{>0}$ は可算）なので、その像である $\mathbb{Q}$ も可算無限集合。

**(d) 整数係数の多項式全体 $\mathbb{Z}[x]$**

- 各多項式は有限個の係数 $(a_0,\dots,a_n)$ で表せる。
- 長さ $n+1$ の整数列全体は $\mathbb{Z}^{n+1}$ と同相で、可算無限集合。
- 可算無限個の可算無限集合の和集合として表せるので、可算無限集合。

__(3) 可算無限集合の部分集合の性質__

$A$ を可算無限集合とし、$B \subset A$ とする。

- $B$ が有限なら有限集合。
- $B$ が無限なら、$A$ から $\mathbb{N}$ への全単射 $f$ をとり、$f(B) \subset \mathbb{N}$ は無限部分集合。
- 自然数の無限部分集合は $\mathbb{N}$ と順序同型（最小元から順に番号を振れる）なので可算無限。
- よって $B$ も可算無限集合。

__問4（非可算無限集合）__

__(1) $\mathbb{R}$ が非可算であること（対角線論法）__

**証明**（背理法）：

- $\mathbb{R}$ が可算無限であると仮定する。すると、$\mathbb{R}$ の元を $r_1, r_2, r_3, \dots$ と列挙できる。
- 各実数 $r_n$ を10進小数展開（一意性のために、有限小数は $0.999\dots$ 型を避ける）で
  $$
  r_n = 0.a_{n1}a_{n2}a_{n3}\dots
  $$
  と表す。
- 新しい実数 $s$ を
  $$
  s = 0.b_1b_2b_3\dots,\quad b_n = \begin{cases}
  1 & (a_{nn} \ne 1) \\
  2 & (a_{nn} = 1)
  \end{cases}
  $$
  と定義する。
- この $s$ はどの $r_n$ とも $n$ 桁目が異なるので、列挙に含まれない。
- よって矛盾。したがって $\mathbb{R}$ は非可算無限集合。

__(2) $(0,1)$ と $\mathbb{R}$ の濃度が等しいこと__

- 関数 $f: (0,1) \to \mathbb{R}$ を
  $$
  f(x) = \tan\left(\pi\left(x-\frac{1}{2}\right)\right)
  $$
  と定義すると、これは全単射（$(0,1)$ を $(-\pi/2,\pi/2)$ に線形写像し、tan で $\mathbb{R}$ 全体に写す）。
- よって $|(0,1)| = |\mathbb{R}| = 2^{\aleph_0}$。

__(3) 連続体濃度 $2^{\aleph_0}$ の例__

**(a) 閉区間 $[0,1]$**

- $(0,1) \subset [0,1] \subset \mathbb{R}$ かつ $|(0,1)| = |\mathbb{R}|$ なので、$|[0,1]| = 2^{\aleph_0}$。

**(b) 複素数全体 $\mathbb{C}$**

- $\mathbb{C} \cong \mathbb{R} \times \mathbb{R}$ なので $|\mathbb{C}| = |\mathbb{R}| \cdot |\mathbb{R}| = 2^{\aleph_0} \cdot 2^{\aleph_0} = 2^{\aleph_0}$。

**(c) 実数列全体 $\mathbb{R}^\mathbb{N}$**

- $|\mathbb{R}^\mathbb{N}| = (2^{\aleph_0})^{\aleph_0} = 2^{\aleph_0 \cdot \aleph_0} = 2^{\aleph_0}$。

__問5（基数の演算）__

__(1) 演算の定義__

基数 $\kappa,\lambda$ に対し、$|A|=\kappa,\ |B|=\lambda$ となる集合 $A,B$ をとる。

- **(a) 和**：$\kappa + \lambda = |A \cup B|$（ただし $A \cap B = \varnothing$）
- **(b) 積**：$\kappa \cdot \lambda = |A \times B|$
- **(c) 冪**：$\kappa^\lambda = |\{f \mid f: B \to A\}|$

__(2) 等式の判定__

**(a) $\aleph_0 + 5 = \aleph_0$**

- 真。無限集合に有限個の元を加えても濃度は変わらない。

**(b) $\aleph_0 + \aleph_0 = \aleph_0$**

- 真。可算無限集合2つの非交和は可算無限。

**(c) $\aleph_0 \cdot \aleph_0 = \aleph_0$**

- 真。$\mathbb{N} \times \mathbb{N}$ は可算無限。

**(d) $2^{\aleph_0} = \aleph_0$**

- 偽。カントールの定理より $2^{\aleph_0} > \aleph_0$。

__(3) $\kappa + \lambda = \kappa \cdot \lambda = \max(\kappa,\lambda)$ の証明__

$\kappa$ を無限基数、$\lambda$ を任意の基数とする。

- $\kappa \le \lambda$ のとき：$\max(\kappa,\lambda) = \lambda$。
  - $\kappa + \lambda \le \lambda + \lambda = 2\cdot\lambda \le \lambda\cdot\lambda = \lambda$（無限基数の性質）。
  - 一方 $\lambda \le \kappa + \lambda$ は明らか。
  - よって $\kappa + \lambda = \lambda$。
  - 同様に $\kappa \cdot \lambda = \lambda$。
- $\lambda \le \kappa$ のとき：$\max(\kappa,\lambda) = \kappa$。
  - $\kappa + \lambda \le \kappa + \kappa = 2\cdot\kappa = \kappa$。
  - 一方 $\kappa \le \kappa + \lambda$ は明らか。
  - よって $\kappa + \lambda = \kappa$。
  - 同様に $\kappa \cdot \lambda = \kappa$。

したがって、いずれの場合も $\kappa + \lambda = \kappa \cdot \lambda = \max(\kappa,\lambda)$。

__問6（アレフ数と連続体仮説）__

__(1) アレフ数の定義__

順序数 $\alpha$ に対して、**アレフ数** $\aleph_\alpha$ は次のように定義される。

- $\aleph_0$：最小の無限基数（可算無限濃度）。
- 順序数 $\alpha$ が与えられたとき、$\aleph_\alpha$ は「それより小さいすべての基数 $\aleph_\beta$（$\beta<\alpha$）と全単射を持たない」最小の順序数。

__(2) アレフ数の性質__

**(a) $\aleph_0$**

- 自然数全体 $\mathbb{N}$ の濃度。可算無限集合の濃度。

**(b) $\aleph_1$**

- $\aleph_0$ の次の基数。最小の非可算基数。

**(c) $\aleph_2$**

- $\aleph_1$ の次の基数。$\aleph_1 < \aleph_2$。

__(3) 連続体仮説（CH）と一般連続体仮説（GCH）__

- **CH**：$2^{\aleph_0} = \aleph_1$
- **GCH**：任意の無限基数 $\kappa$ について $2^\kappa = \kappa^+$

これらは ZFC から独立（証明も反証もできない）であることが知られています。

__問7（濃度の同値類）__

__(1) 同値関係の well-defined 性__

「濃度が等しい」という関係を

- $A \sim B \iff $ $A$ と $B$ の間に全単射が存在する

と定義する。

- 反射律：恒等写像により $A \sim A$。
- 対称律：$A \sim B$ なら全単射 $f: A \to B$ が存在し、その逆写像 $f^{-1}: B \to A$ も全単射なので $B \sim A$。
- 推移律：$A \sim B$ かつ $B \sim C$ なら、全単射の合成により $A \sim C$。

よって同値関係であり、商集合が well-defined。

__(2) 大小関係 $\le$ が半順序であること__

- 反射的：恒等写像により $|A| \le |A|$。
- 反対称：（ベルンシュタインの定理）$|A| \le |B|$ かつ $|B| \le |A|$ なら $|A| = |B|$。
- 推移的：単射の合成により $|A| \le |B|$ かつ $|B| \le |C|$ なら $|A| \le |C|$。

__(3) 全順序性に依存する公理__

濃度の大小関係が全順序（任意の2つの濃度が比較可能）であることは、**選択公理（AC）** に依存します。

- AC を仮定すると、任意の集合は整列可能であり、順序数と比較できるため濃度も比較可能。
- AC を仮定しない場合、比較不能な濃度が存在しうることが知られています。

__問8（基数の順序数としての性質）__

__(1) 基数の定義（順序数として）__

順序数 $\alpha$ が**基数**であるとは、

- 任意の $\beta < \alpha$ について $|\beta| \ne |\alpha|$

が成り立つことをいう。すなわち、$\alpha$ は「自分より小さい順序数との間に全単射が存在しない」最小の順序数。

__(2) 最小の順序数の存在__

任意の集合 $A$ に対し、整列可能定理（AC と同値）により $A$ を整列できる。

- 整列順序 $(A,\le)$ の順序型を $\alpha$ とすると、$A$ は順序数 $\alpha$ と順序同型。
- 順序数全体は整列クラスなので、$A$ と順序同型な順序数のうち最小のものが存在する。
- これを $A$ の基数と定義する。

__(3) $\omega$ が基数であること__

- $\omega$ は有限順序数 $0,1,2,\dots$ より大きいが、有限順序数とは全単射を持たない（有限と無限は濃度が異なる）。
- よって $\omega$ は基数である（$\aleph_0$）。

__問9（正則基数と特異基数）__

__(1) 正則基数の定義__

基数 $\kappa$ が**正則**であるとは、

- 任意の集合族 $\{A_\alpha\}_{\alpha<\lambda}$ について、
  - $|A_\alpha| < \kappa$（各 $\alpha<\lambda$）
  - $\lambda < \kappa$
- ならば
  - $\left|\bigcup_{\alpha<\lambda} A_\alpha\right| < \kappa$

が成り立つことをいう。

__(2) 共終数との同値性__

基数 $\kappa$ の**共終数** $\mathrm{cf}(\kappa)$ は、

- $\kappa$ に共終な増加列の最小の長さ

として定義される。

このとき、

- $\kappa$ が正則 ⇔ $\mathrm{cf}(\kappa) = \kappa$
- $\kappa$ が特異 ⇔ $\mathrm{cf}(\kappa) < \kappa$

が成り立つ。

__(3) 正則・特異の判定__

**(a) $\aleph_0$**

- 正則。有限個または可算個の有限集合の和は高々可算。

**(b) $\aleph_1$**

- 正則。後続基数は常に正則。

**(c) $\aleph_\omega$**

- 特異。$\mathrm{cf}(\aleph_\omega) = \omega < \aleph_\omega$。

**(d) $\aleph_{\omega+1}$**

- 正則。後続基数。

__(4) 直感的な違い__

- 正則基数：小さな集合を小さな個数だけ集めても、全体はその基数に達しない。
- 特異基数：小さな集合を小さな個数だけ集めると、全体がその基数になってしまう。

__問10（基数演算と正則・特異基数）__

__(1) 正則基数における和__

$\kappa$ を正則基数、$\lambda < \kappa$ とする。

- $\kappa + \lambda = \max(\kappa,\lambda) = \kappa$（無限基数の和の性質）。
- 正則性から、$\kappa$ 未満の濃度を $\kappa$ 未満個集めても $\kappa$ に達しないが、ここでは演算の定義により直接 $\kappa + \lambda = \kappa$ が得られる。

__(2) 特異基数の「和としての表現」__

$\kappa$ が特異なら $\mathrm{cf}(\kappa) = \lambda < \kappa$。

- 共終な増加列 $\{\kappa_\alpha\}_{\alpha<\lambda}$（各 $\kappa_\alpha < \kappa$）が存在し、
  $$
  \kappa = \sup_{\alpha<\lambda} \kappa_\alpha
  $$
- この列を用いると、$\kappa$ は「より小さい基数 $\kappa_\alpha$ を $\lambda$ 個集めたもの」として表現できる。

__(3) 演算結果と正則・特異__

**(a) $\aleph_\omega + \aleph_0$**

- $= \max(\aleph_\omega, \aleph_0) = \aleph_\omega$（特異）。

**(b) $\aleph_\omega \cdot \aleph_0$**

- $= \max(\aleph_\omega, \aleph_0) = \aleph_\omega$（特異）。

**(c) $\aleph_1 + \aleph_\omega$**

- $= \max(\aleph_1, \aleph_\omega) = \aleph_\omega$（特異）。

**(d) $\aleph_1 \cdot \aleph_\omega$**

- $= \max(\aleph_1, \aleph_\omega) = \aleph_\omega$（特異）。

__問11（連続体仮説の応用）__

__(1) CH 下での命題の真偽__

CH を仮定すると $2^{\aleph_0} = \aleph_1$。

- 実数の非可算部分集合 $X \subset \mathbb{R}$ は $|X| \ge \aleph_1$。
- 一方 $|X| \le |\mathbb{R}| = \aleph_1$。
- よって $|X| = \aleph_1 = 2^{\aleph_0}$。

したがって命題は真。

__(2) CH を仮定しない場合の可能性__

CH を仮定しない ZFC では、$2^{\aleph_0}$ は $\aleph_1$ より大きくなり得る。

- このとき、濃度が $\aleph_1$ で $< 2^{\aleph_0}$ となる実数部分集合が存在しうる。
- よって「非可算なら連続体濃度」とは限らない。

__(3) CH の世界と非 CH の世界の違い__

- CH が成り立つ世界：実数の部分集合の濃度は
  - 可算（$\aleph_0$）
  - 連続体濃度（$\aleph_1$）
  の2種類のみ。
- CH が成り立たない世界：中間的な濃度（例：$\aleph_1$）の部分集合が存在しうる。

__問12（巨大基数と連続体仮説）__

__(1) 到達不能基数の定義__

基数 $\kappa$ が**到達不能（inaccessible）** であるとは、次の3条件を満たすことをいう。

1. **非可算**：$\kappa > \aleph_0$
2. **正則**：$\mathrm{cf}(\kappa) = \kappa$
3. **強極限**：任意の $\lambda < \kappa$ について $2^\lambda < \kappa$

__(2) 到達不能基数が正則であること__

定義により正則性（$\mathrm{cf}(\kappa) = \kappa$）は明示的に要求されている。

__(3) 到達不能基数の存在の独立性__

- 到達不能基数の存在は ZFC から証明できない（ZFC が無矛盾なら、ZFC +「到達不能基数は存在しない」も無矛盾）。
- また、ZFC +「到達不能基数が存在する」も無矛盾（巨大基数公理として採用可能）。
- よってその存在は ZFC から独立。

__(4) 巨大基数公理と CH の関係__

- 巨大基数公理（到達不能基数以上の存在）を仮定しても、CH の真偽は決定されない。
- 巨大基数公理は強制法によるモデル構成を豊かにし、CH が成り立たないモデルを構成しやすくするが、CH を「必然的に偽」にするわけではない。
- CH の真偽は、巨大基数公理とは独立に選択できる追加公理として扱われる。
