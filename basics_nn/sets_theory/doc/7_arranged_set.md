# 整列集合

## 整列順序

**整列集合（well-ordered set）** は、全順序集合のうち、**任意の空でない部分集合が最小元を持つ**という強い条件を満たすものです。  
順序数（ordinal numbers）の理論や超限帰納法の基礎となる重要な概念です。

### 1. 整列集合の定義

全順序集合 $(X,\leq)$ が**整列集合**であるとは、次の条件を満たすことをいいます。

> 任意の空でない部分集合 $A \subset X$ は、**最小元**を持つ。

すなわち、$A \neq \varnothing$ ならば、ある $m \in A$ が存在して、すべての $a \in A$ について $m \leq a$ が成り立つ。

このとき、$\leq$ を**整列順序（well-order）** と呼びます。

### 2. 定義の注意点

__(1) 全順序であること__

- 整列集合は**全順序集合**の一種です。
- 部分順序集合では「比較不能な元」が存在し得ますが、整列集合では任意の2元が比較可能です。
- したがって、最小元の一意性は自動的に保証されます（全順序性＋反対称律より）。

__(2) 「任意の空でない部分集合」が最小元を持つ__

- 有限部分集合だけでなく、**無限部分集合**も含めて、すべての空でない部分集合が最小元を持たなければなりません。
- この条件は、**降鎖条件（descending chain condition）** とも関係し、「無限に小さくなり続ける列」が存在しないことを意味します。

### 3. 整列集合の具体例

__(1) 有限の全順序集合__

- 例： $(\{1,2,3\},\leq)$ 、$(\{a,b,c\},\leq)$（辞書順）など。
- 有限集合では、任意の空でない部分集合は有限個の元からなるので、その中で最小の元が必ず存在します。
- したがって、**有限の全順序集合はすべて整列集合**です。

__(2) 自然数全体 $(\mathbb{N},\leq)$__

- 任意の空でない自然数の集合には、最小の自然数が存在します（自然数の整序性）。
- 例：偶数全体 $\{2,4,6,\dots\}$ の最小元は 2。
- よって $(\mathbb{N},\leq)$ は整列集合です。

__(3) 順序数（ordinal numbers）__

- 順序数そのものが整列集合として定義されます。
- 例： $\omega$ （自然数型）、 $\omega+1$ 、 $\omega\cdot2$ などはすべて整列集合です。

### 4. 整列集合でない例

__(1) 整数全体 $(\mathbb{Z},\leq)$__

- 部分集合 $A = \{\dots,-3,-2,-1\}$（負の整数全体）を考えると、$A$ には最小元が存在しません（いくらでも小さい負の整数がある）。
- よって $(\mathbb{Z},\leq)$ は整列集合ではありません。

__(2) 有理数全体 $(\mathbb{Q},\leq)$__

- 部分集合 $A = \{x \in \mathbb{Q} \mid x > 0\}$ を考えると、$A$ には最小元が存在しません（任意の正の有理数より小さい正の有理数が存在する）。
- よって $(\mathbb{Q},\leq)$ は整列集合ではありません。

__(3) 実数全体 $(\mathbb{R},\leq)$__

- 同様に、正の実数全体には最小元が存在しないので、整列集合ではありません。
- 実は、**選択公理**を用いると「任意の集合に整列順序が存在する（整列可能定理）」ことが示されますが、その整列順序は通常の大小関係とは異なり、具体的に構成できるとは限りません。

### 5. 整列集合の基本的性質

__(1) 整列集合の部分集合__

- 整列集合の任意の部分集合は、制限された順序に関して再び整列集合になります。
- 特に、**整列集合の始片（initial segment）** は整列集合です。

__(2) 整列集合の順序同型類：順序数__

- 2つの整列集合が順序同型であるとき、それらは同じ**順序型**を持ちます。
- 整列集合の順序同型類を**順序数（ordinal number）** と呼びます。
- 例：$\mathbb{N}$ の順序型は $\omega$、$\{1,2,\dots,n\}$ の順序型は自然数 $n$ です。

__(3) 整列集合の比較定理__

- 任意の2つの整列集合は、一方が他方の始片と順序同型であるか、または互いに順序同型である、という強い性質を持ちます。
- これにより、順序数には自然な全順序 $\leq$ が入り、「順序数の大小」が定義されます。


### 6. なぜ整列集合が必要なのか

__(1) 順序数理論の基礎__

- 順序数は「整列集合の順序型」として定義され、無限の大きさを測る基本的な道具です。
- 集合論の多くの結果（基数の比較、累積階層など）は順序数に依存しています。

__(2) 超限帰納法・超限再帰__

- 無限の構造に対しても帰納法や再帰的定義を適用できるようにします。
- 例：集合論の定理の証明、再帰的関数の定義、木構造の解析など。

__(3) 整列可能定理と選択公理__

- **整列可能定理**（任意の集合に整列順序が存在する）は、**選択公理**と同値です。
- これは「任意の集合を一直線に並べられる」ことを主張し、集合論の基礎的な結果の一つです。

## 直後要素、直前要素、極限要素

**直後要素（immediate successor）**・**直前要素（immediate predecessor）**・**極限要素（limit element）**は、主に**整列集合**や**順序数**の文脈で使われる概念です。  
これらは、順序集合の中で「すぐ次に来る元」「すぐ前に来る元」「それより小さい元たちの上限として現れる元」を表します。

以下、順序集合 $(X,\leq)$ に対して定義します。

### 1. 直後要素（immediate successor）

__定義__

元 $x \in X$ に対して、**直後要素**とは、次の条件を満たす元 $y \in X$ のことです。

1. $x < y$（すなわち $x \leq y$ かつ $x \neq y$）。
2. $x < z < y$ となる $z \in X$ が存在しない。

言い換えると、$y$ は $x$ より大きく、かつ $x$ と $y$ の間に他の元が入らない「すぐ次の元」です。

- 記号：$y = x^+$ などと書くことがあります。
- 存在すれば**一意**です（全順序性などから）。

__具体例__

- $(\mathbb{N},\leq)$：  
  - 任意の自然数 $n$ に対して、直後要素は $n+1$。
- $(\mathbb{Z},\leq)$：  
  - 任意の整数 $k$ に対して、直後要素は $k+1$。
- 有限全順序集合：  
  - 最大元以外の各元に直後要素が存在する。
- $(\mathbb{Q},\leq)$ や $(\mathbb{R},\leq)$：  
  - 稠密なので、任意の元 $x$ に対して $x < y$ となる $y$ は存在するが、$x$ と $y$ の間に必ず他の有理数・実数が存在する。  
  - よって、**直後要素は存在しない**。

### 2. 直前要素（immediate predecessor）

__定義__

元 $x \in X$ に対して、**直前要素**とは、次の条件を満たす元 $y \in X$ のことです。

1. $y < x$（すなわち $y \leq x$ かつ $y \neq x$）。
2. $y < z < x$ となる $z \in X$ が存在しない。

言い換えると、$y$ は $x$ より小さく、かつ $y$ と $x$ の間に他の元が入らない「すぐ前の元」です。

- 記号：$y = x^-$ などと書くことがあります。
- 存在すれば**一意**です。

__具体例__

- $(\mathbb{N},\leq)$：  
  - 1 には直前要素がない（最小元）。  
  - $n \geq 2$ に対して、直前要素は $n-1$。
- $(\mathbb{Z},\leq)$：  
  - 任意の整数 $k$ に対して、直前要素は $k-1$。
- 有限全順序集合：  
  - 最小元以外の各元に直前要素が存在する。
- $(\mathbb{Q},\leq)$ や $(\mathbb{R},\leq)$：  
  - 稠密なので、**直前要素は存在しない**。

### 3. 極限要素（limit element）

**極限要素**は、主に**順序数**や**整列集合**の文脈で定義されます。  
直感的には、「それより小さい元たちの『極限』として現れる元」です。

__定義（順序数の文脈）__

順序数 $\alpha$ が**極限順序数（limit ordinal）** であるとは、次の条件を満たすことをいいます。

- 直前要素が存在しない。すなわち、$\alpha = \beta + 1$ となる順序数 $\beta$ が存在しない。

ここで、$\beta + 1$ は $\beta$ の直後順序数（$\beta$ に新しい最大元を付け加えた順序数）です。

より一般的に、整列集合 $(X,\leq)$ において、元 $x \in X$ が**極限要素**であるとは：

- $x$ は最小元ではない。
- $x$ に直前要素が存在しない。

すなわち、$x$ は「ある元の直後」としてではなく、それより小さい元たちの**上限**として現れる元です。

__具体例（順序数）__

- $\omega$（自然数全体の順序型）：  
  - 任意の自然数 $n$ に対して $n < \omega$ だが、$\omega$ の直前要素は存在しない（自然数は有限で、$\omega$ は無限）。  
  - よって $\omega$ は極限順序数。
- $\omega + 1$：  
  - 直前要素は $\omega$ なので、極限順序数ではない（**後続順序数**）。
- $\omega \cdot 2 = \omega + \omega$：  
  - 直前要素は存在しないので極限順序数。
- 0 や 1,2,3,... は極限順序数ではない（0 は最小、1=0+1, 2=1+1, ... はすべて後続順序数）。

__具体例（他の整列集合）__

- $(\mathbb{N},\leq)$：  
  - 最小元 1 以外はすべて直前要素を持つ（$n$ の直前は $n-1$）。  
  - よって極限要素は存在しない。
- 順序数 $\omega$ に対応する整列集合（例：$\{0,1,2,\dots\}$）では、  
  - 任意の有限順序数 $n$ には直前要素があるが、$\omega$ 自体には直前要素がない → $\omega$ が極限要素。

### 4. 後続順序数（successor ordinal）との対比

- **後続順序数**：直前要素を持つ順序数。すなわち $\alpha = \beta + 1$ と書けるもの。
- **極限順序数**：直前要素を持たない 0 でない順序数。

任意の 0 でない順序数は、**後続順序数**か**極限順序数**のどちらかです。

### 5. なぜこれらの概念が必要か

__(1) 順序数の分類__

- 順序数を「0」「後続順序数」「極限順序数」の3種類に分けることで、超限帰納法や超限再帰の定義がきれいに書けます。
- 例：超限帰納法の段階で、
  - 基底：0 に対する証明。
  - 後続段階：$\alpha = \beta+1$ のとき、$\beta$ での成立から $\alpha$ での成立を示す。
  - 極限段階：$\alpha$ が極限順序数のとき、すべての $\beta < \alpha$ での成立から $\alpha$ での成立を示す。

__(2) 無限の構造の理解__

- 極限順序数は、「有限ステップでは到達できないが、無限のプロセスの極限として現れる」ような元を表します。
- 例：$\omega$ は有限回の「+1」操作では到達できないが、すべての自然数の極限として現れる。

__(3) 集合論・モデル理論での応用__

- 巨大な順序数（到達不能順序数など）は、極限順序数として定義され、集合論のモデルや巨大基数の理論で重要な役割を果たします。
- 直後・直前・極限の概念は、順序構造の「局所的な形」を分析するのに役立ちます。


## 超限帰納法

**超限帰納法（transfinite induction）**は、数学的帰納法を**整列集合**や**順序数**上に拡張した証明法です。  
「すべての順序数（または整列集合の元）について命題が成り立つ」ことを示すために用いられます。

以下、順序数 $\mathrm{Ord}$ 上の超限帰納法を中心に説明しますが、一般の整列集合でも同様に定義できます。

### 1. 超限帰納法の定義（順序数版）

順序数全体のクラス $\mathrm{Ord}$ に対して、命題 $P(\alpha)$（$\alpha$ は順序数）が与えられているとします。

超限帰納法とは、次の推論規則です。

> 次の2条件が成り立つならば、**すべての順序数 $\alpha$ について $P(\alpha)$ が成り立つ**。
>
> 1. **基底（base case）**：  
>    $P(0)$ が成り立つ。
> 2. **帰納段階（inductive step）**：  
>    任意の順序数 $\alpha$ について、  
>    「すべての $\beta < \alpha$ に対して $P(\beta)$ が成り立つならば、$P(\alpha)$ も成り立つ」  
>    ことを示す。

ここで、$\beta < \alpha$ は順序数の大小関係（整列順序）です。

__直感的な意味__

- 通常の数学的帰納法は「$P(n)$ から $P(n+1)$ を示す」ですが、超限帰納法では「**$\alpha$ より小さいすべての順序数で成り立てば、$\alpha$ でも成り立つ**」ことを要求します。
- これにより、有限のステップでは到達できない極限順序数（例：$\omega$）についても命題を示すことができます。

### 2. 3段階に分けた書き方（後続・極限の場合分け）

順序数は次の3種類に分けられます。

- **0**：最小の順序数。
- **後続順序数（successor ordinal）**：$\alpha = \beta + 1$ と書けるもの。
- **極限順序数（limit ordinal）**：0 ではなく、かつ後続順序数でないもの（直前要素を持たない）。

これに対応して、超限帰納法は次の3段階で記述されることが多いです。

> すべての順序数 $\alpha$ について $P(\alpha)$ が成り立つことを示すには、次を示せばよい。
>
> 1. **基底**：$P(0)$ を示す。
> 2. **後続段階**：任意の順序数 $\beta$ について、$P(\beta)$ が成り立つならば $P(\beta+1)$ も成り立つことを示す。
> 3. **極限段階**：任意の極限順序数 $\lambda$ について、  
>    「すべての $\beta < \lambda$ に対して $P(\beta)$ が成り立つならば $P(\lambda)$ も成り立つ」  
>    ことを示す。

__なぜこの3段階で十分か__

- 任意の 0 でない順序数は、後続順序数か極限順序数のどちらかです。
- 後続段階で「$\beta \to \beta+1$」の推移が保証され、極限段階で「より小さい順序数全体から極限順序数へのジャンプ」が保証されます。
- 基底から始めて、後続段階と極限段階を繰り返すことで、すべての順序数にわたって $P(\alpha)$ が成立することが示されます（整列性による）。

### 3. 一般の整列集合上の超限帰納法

整列集合 $(X,\leq)$ に対して、命題 $P(x)$（$x \in X$）が与えられているとします。

超限帰納法は次のように定式化されます。

> 次の条件が成り立つならば、**すべての $x \in X$ について $P(x)$ が成り立つ**。
>
> 任意の $x \in X$ について、  
> 「すべての $y < x$ に対して $P(y)$ が成り立つならば、$P(x)$ も成り立つ」  
> ことを示せ。

ここで $y < x$ は $y \leq x$ かつ $y \neq x$ を意味します。

__基底の扱い__

- 最小元 $m \in X$ が存在する場合、$y < m$ となる $y$ は存在しないので、  
  「すべての $y < m$ に対して $P(y)$」は vacuously true（空虚な真）です。
- したがって、上記の条件から自動的に $P(m)$ が導かれます。  
  明示的に基底 $P(m)$ を示す必要はありませんが、実際の証明では通常 $P(m)$ を別途確認します。

### 4. 超限帰納法の具体例

__例1：順序数の加法の結合則の証明（概略）__

順序数の加法 $\alpha + \beta$ が定義されているとき、  
$(\alpha + \beta) + \gamma = \alpha + (\beta + \gamma)$ をすべての $\alpha,\beta,\gamma$ について示したい。

- 固定した $\alpha,\beta$ に対して、$\gamma$ に関する超限帰納法を用いる。
- 基底：$\gamma = 0$ のとき、左辺＝$\alpha+\beta$、右辺＝$\alpha+\beta$ で成立。
- 後続段階：$\gamma = \delta+1$ のとき、帰納仮定（$\delta$ での成立）から $\gamma$ での成立を示す。
- 極限段階：$\gamma$ が極限順序数のとき、$\gamma = \sup_{\delta<\gamma} \delta$ を用いて、両辺の極限として成立を示す。

（詳細は順序数の定義に依存しますが、枠組みとして超限帰納法が使われます。）

__例2：整列集合の始片の性質__

整列集合 $(X,\leq)$ の始片（initial segment）が常に整列集合であることを示すのに、超限帰納法が使えます。

- $P(x)$：「$x$ より小さい元全体の集合 $\{y \in X \mid y < x\}$ は整列集合である」とおく。
- 基底：最小元 $m$ について、$\{y \mid y < m\} = \varnothing$ は（自明に）整列集合。
- 帰納段階：任意の $x$ について、すべての $y < x$ で $P(y)$ が成り立つと仮定し、$P(x)$ を示す。  
  （ここで $x$ が後続か極限かで場合分けが必要になることが多い。）

### 5. 超限帰納法の正当性（なぜ成り立つか）

超限帰納法が正当である理由は、**整列集合の定義**（任意の空でない部分集合が最小元を持つ）にあります。

背理法で示すことができます。

- ある $x \in X$ で $P(x)$ が偽になると仮定する。
- $A = \{x \in X \mid \lnot P(x)\}$ とおくと、$A$ は空でない。
- 整列性より $A$ には最小元 $m$ が存在する。
- すると、すべての $y < m$ について $P(y)$ は真（$m$ が偽の最小元だから）。
- しかし帰納段階の仮定より、このとき $P(m)$ も真でなければならない → 矛盾。

よって、そのような $x$ は存在せず、すべての $x$ で $P(x)$ が真です。

### 6. 超限帰納法と超限再帰

超限帰納法に対応する「定義の方法」として、**超限再帰（transfinite recursion）** があります。

- 超限帰納法：命題 $P(\alpha)$ がすべての順序数で成り立つことを示す。
- 超限再帰：関数 $F(\alpha)$ を「より小さい順序数での値」に基づいて定義する。

例：順序数の加法 $\alpha + \beta$ は、超限再帰によって定義されます。

__例題:__

以下、超限帰納法を用いる典型的な例題を出題します。  


__1. 整列集合の始片の整列性__

整列集合 $(X,\leq)$ の**始片（initial segment）**を
$$
X_{<x} = \{y \in X \mid y < x\}
$$
と定義する。

次の命題を**超限帰納法**（または整列集合上の帰納法）を用いて証明せよ。

> 任意の $x \in X$ について、$(X_{<x},\leq)$ は整列集合である。

（ヒント：$P(x)$：「$X_{<x}$ は整列集合である」とおき、$x$ に関する帰納法を用いる。）

---

以下、整列集合 $(X,\leq)$ の始片 $X_{<x}$ が整列集合であることを、**超限帰納法**を用いて証明します。

---

__証明：整列集合の始片の整列性__

__1. 記号と目標__

- $(X,\leq)$：整列集合（全順序かつ任意の空でない部分集合が最小元を持つ）。
- 始片の定義：
  $$
  X_{<x} = \{ y \in X \mid y < x \}
  $$
  （$y < x$ は $y \leq x$ かつ $y \neq x$ を意味する）。
- 目標：任意の $x \in X$ について、$(X_{<x},\leq)$ が整列集合であることを示す。

__2. 超限帰納法の設定__

整列集合 $(X,\leq)$ 上で、命題 $P(x)$ を次のように定義する。

$$
P(x): \quad \text{「}(X_{<x},\leq)\text{ は整列集合である」}
$$

超限帰納法（整列集合上の帰納法）の原理より、次を示せばよい。

> 任意の $x \in X$ について、  
> 「すべての $y < x$ に対して $P(y)$ が成り立つならば、$P(x)$ も成り立つ」

この条件が示されれば、整列性によりすべての $x \in X$ で $P(x)$ が成り立つ。

__3. 帰納段階の証明__

任意の $x \in X$ を固定し、次の仮定をおく。

- **帰納仮定**：すべての $y < x$ について、$(X_{<y},\leq)$ は整列集合である。

このとき、$(X_{<x},\leq)$ が整列集合であることを示す。

__(a) 全順序性の確認__

- $X_{<x} \subset X$ であり、$X$ 上の順序 $\leq$ は全順序（任意の2元が比較可能）。
- 部分集合に制限した順序も同じ比較規則なので、$(X_{<x},\leq)$ も全順序集合である。

__(b) 任意の空でない部分集合が最小元を持つこと__

$A \subset X_{<x}$ を任意の空でない部分集合とする。  
$A \subset X_{<x} \subset X$ なので、$A$ は $X$ の空でない部分集合でもある。

- $(X,\leq)$ は整列集合なので、$A$ は $X$ において最小元を持つ。  
  その最小元を $m$ とおく：
  $$
  m \in A,\quad \forall a \in A,\ m \leq a
  $$
- $A \subset X_{<x}$ より、すべての $a \in A$ について $a < x$。  
  特に $m < x$ なので、$m \in X_{<x}$。
- したがって、$m$ は $A$ の（$X_{<x}$ における）最小元でもある。

以上より、$(X_{<x},\leq)$ の任意の空でない部分集合 $A$ は最小元を持つ。

__(c) 結論__

(a) より $(X_{<x},\leq)$ は全順序集合であり、(b) より任意の空でない部分集合が最小元を持つ。  
よって $(X_{<x},\leq)$ は整列集合である。すなわち $P(x)$ が成り立つ。

__4. 超限帰納法の適用__

- 任意の $x \in X$ について、「すべての $y < x$ で $P(y)$ が成り立つならば $P(x)$ が成り立つ」ことを示した。
- 整列集合上の超限帰納法の原理により、**すべての $x \in X$ について $P(x)$ が成り立つ**。

したがって、任意の $x \in X$ について $(X_{<x},\leq)$ は整列集合である。

__まとめ__

- 整列集合 $(X,\leq)$ の始片 $X_{<x}$ は、$X$ の部分集合として同じ順序を引き継ぐので全順序。
- 任意の空でない部分集合 $A \subset X_{<x}$ は $X$ の空でない部分集合でもあり、整列性より最小元 $m$ を持つ。
- $m \in A \subset X_{<x}$ より $m < x$ なので、$m$ は $X_{<x}$ における最小元。
- よって $(X_{<x},\leq)$ は整列集合。
- この事実を命題 $P(x)$ とおき、超限帰納法によりすべての $x \in X$ で成立することを示した。

### 解答

以下、各問の解答です。

__問1（整列集合の定義と具体例）__

__(1) 整列集合かどうかの判定__

**(a) $\{1,2,3\}$（通常の大小）**

- 整列集合である。
- 理由：有限全順序集合は、任意の空でない部分集合が有限個の元からなり、その中で最小元が必ず存在するため。

**(b) $\mathbb{Z}$（整数全体、通常の大小）**

- 整列集合ではない。
- 理由：部分集合 $A = \{\dots,-3,-2,-1\}$（負の整数全体）を考えると、$A$ は空でないが最小元を持たない（いくらでも小さい負の整数がある）。

**(c) $\{x \in \mathbb{Q} \mid x > 0\}$（正の有理数全体、通常の大小）**

- 整列集合ではない。
- 理由：任意の正の有理数 $q > 0$ に対して、$0 < q/2 < q$ となる有理数が存在するため、$A$ 全体として最小元を持たない。

**(d) $\{0\} \cup \{1/n \mid n \in \mathbb{N}\}$（0 と 1, 1/2, 1/3, …、通常の大小）**

- 整列集合である。
- 理由：
  - 全順序である（通常の実数の大小で比較可能）。
  - 任意の空でない部分集合 $A$ について：
    - もし $0 \in A$ なら、$0$ が最小元。
    - もし $0 \notin A$ なら、$A$ は $\{1/n\}$ の部分集合であり、その中で最大の $1/n$ が $A$ の最小元になる（$1/n$ が最大なら、通常の大小では最小になることに注意：$1/1 > 1/2 > 1/3 > \dots$）。
  - よって、任意の空でない部分集合に最小元が存在する。

__(2) 「全順序であること」が必要な理由__

- 整列集合は「任意の空でない部分集合が最小元を持つ」ことを要求するが、**最小元の一意性**を保証するには、任意の2元が比較可能である必要がある。
- もし部分順序集合で「比較不能な元」が存在すると、最小元が複数存在し得る（例：ハッセ図で枝分かれしている場合）。
- 例：集合 $X = \{a,b\}$ に順序を
  - $a \le a,\ b \le b$ のみとし、$a$ と $b$ は比較不能とする。
  - このとき $X$ 自体は空でないが、「最小元」は $a$ と $b$ の両方が候補になり、一意に定まらない。
- 全順序であれば、反対称律と合わせて最小元の一意性が保証されるため、整列集合の定義に「全順序であること」が含まれる。

__問3（超限帰納法の理解）__

__(1) 順序数全体 $\mathrm{Ord}$ 上の超限帰納法__

**(a) 2条件での主張**

> 命題 $P(\alpha)$（$\alpha$ は順序数）が次を満たすならば、すべての順序数 $\alpha$ について $P(\alpha)$ が成り立つ。
>
> 1. **基底**：$P(0)$ が成り立つ。
> 2. **帰納段階**：任意の順序数 $\alpha$ について、  
>    「すべての $\beta < \alpha$ に対して $P(\beta)$ が成り立つならば、$P(\alpha)$ も成り立つ」ことを示す。

**(b) 3段階（0・後続・極限）の形**

> すべての順序数 $\alpha$ について $P(\alpha)$ が成り立つことを示すには、次を示せばよい。
>
> 1. **基底**：$P(0)$ を示す。
> 2. **後続段階**：任意の順序数 $\beta$ について、$P(\beta)$ が成り立つならば $P(\beta+1)$ も成り立つことを示す。
> 3. **極限段階**：任意の極限順序数 $\lambda$ について、  
>    「すべての $\beta < \lambda$ に対して $P(\beta)$ が成り立つならば $P(\lambda)$ も成り立つ」  
>    ことを示す。

**(c) なぜ3段階で十分か（直感的説明）**

- 任意の 0 でない順序数は、後続順序数か極限順序数のどちらかである。
- 基底 $P(0)$ から出発し、
  - 後続段階で「$\beta$ で成り立てば $\beta+1$ でも成り立つ」ことを繰り返すことで、すべての後続順序数に命題が伝播する。
  - 極限段階で「$\lambda$ より小さいすべての順序数で成り立てば $\lambda$ でも成り立つ」ことを保証することで、極限順序数にも命題が及ぶ。
- 順序数全体は整列集合（より正確には整列クラス）なので、この3段階を組み合わせることで、すべての順序数について $P(\alpha)$ が成立することが示される。

__(2) 「すべての順序数 $\alpha$ は推移的集合である」の証明方針__

- 目標：任意の順序数 $\alpha$ について、$\beta \in \alpha$ かつ $\gamma \in \beta$ ならば $\gamma \in \alpha$ が成り立つことを示す。
- 方針：$\alpha$ に関する超限帰納法を用いる。
  - **基底**：$\alpha = 0$ のとき、$0 = \varnothing$ は空虚に推移的（含む元がないので条件は自動的に成立）。
  - **帰納段階**：任意の順序数 $\alpha$ を固定し、「すべての $\delta < \alpha$ について $\delta$ は推移的」と仮定して、$\alpha$ が推移的であることを示す。
    - $\beta \in \alpha$ かつ $\gamma \in \beta$ とする。
    - 順序数の定義により、$\beta < \alpha$ なので、帰納仮定より $\beta$ は推移的：$\gamma \in \beta$ かつ $\beta \in \alpha$ なら $\gamma \in \alpha$ が成り立つ（順序数は推移的集合として定義されることが多いが、ここではその事実自体を帰納法で示す流れ）。
  - 以上より、すべての順序数 $\alpha$ について $\alpha$ は推移的集合である。

__問4（整列集合の始片の整列性）__

__(1) 超限帰納法による証明__

整列集合 $(X,\leq)$ の始片を
$$
X_{<x} = \{ y \in X \mid y < x \}
$$
と定義する。

**命題**：任意の $x \in X$ について、$(X_{<x},\leq)$ は整列集合である。

**証明**：

- $P(x)$：「$(X_{<x},\leq)$ は整列集合である」と定義する。
- 超限帰納法（整列集合上の帰納法）により、次を示せばよい：
  > 任意の $x \in X$ について、  
  > 「すべての $y < x$ に対して $P(y)$ が成り立つならば、$P(x)$ も成り立つ」

**帰納段階**：

- 任意の $x \in X$ を固定し、帰納仮定として「すべての $y < x$ について $(X_{<y},\leq)$ は整列集合である」と仮定する。
- $(X_{<x},\leq)$ が整列集合であることを示す。

**(a) 全順序性**

- $X_{<x} \subset X$ であり、$X$ 上の順序 $\leq$ は全順序。
- 部分集合に制限した順序も同じ比較規則なので、$(X_{<x},\leq)$ も全順序集合である。

**(b) 任意の空でない部分集合が最小元を持つこと**

- $A \subset X_{<x}$ を任意の空でない部分集合とする。
- $A \subset X_{<x} \subset X$ より、$A$ は $X$ の空でない部分集合でもある。
- $(X,\leq)$ は整列集合なので、$A$ は $X$ において最小元 $m$ を持つ：
  $$
  m \in A,\quad \forall a \in A,\ m \leq a
  $$
- $A \subset X_{<x}$ より、すべての $a \in A$ について $a < x$。特に $m < x$ なので $m \in X_{<x}$。
- よって、$m$ は $A$ の（$X_{<x}$ における）最小元でもある。

**(c) 結論**

- (a) より $(X_{<x},\leq)$ は全順序集合であり、(b) より任意の空でない部分集合が最小元を持つ。
- したがって $(X_{<x},\leq)$ は整列集合、すなわち $P(x)$ が成り立つ。

**超限帰納法の適用**：

- 任意の $x \in X$ について、「すべての $y < x$ で $P(y)$ が成り立つならば $P(x)$ が成り立つ」ことを示した。
- 整列集合上の超限帰納法の原理により、すべての $x \in X$ について $P(x)$ が成り立つ。
- よって、任意の $x \in X$ について $(X_{<x},\leq)$ は整列集合である。

__(2) 「$A$ が最小元を持つ」部分の説明（自分の言葉）__

- $X_{<x}$ の空でない部分集合 $A$ を考えると、$A$ は $X$ の空でない部分集合でもある。
- $X$ は整列集合なので、$A$ は $X$ において最小元 $m$ を持つ。
- $A \subset X_{<x}$ なので、$m$ も $X_{<x}$ に属し、$X_{<x}$ における順序は $X$ の順序の制限なので、$m$ は $A$ の $X_{<x}$ における最小元でもある。
- したがって、$X$ の整列性を「借りて」$X_{<x}$ の整列性を示している。

__問5（整列部分集合の整列性）__

整列集合 $(X,\leq)$ の任意の部分集合 $Y \subset X$ は、制限された順序に関して整列集合であることを示す。

__(1) 直接証明__

- $Y$ は $X$ の部分集合であり、順序は $\leq$ の制限なので、$(Y,\leq)$ は全順序集合である。
- $A \subset Y$ を任意の空でない部分集合とする。$A \subset Y \subset X$ より、$A$ は $X$ の空でない部分集合でもある。
- $(X,\leq)$ は整列集合なので、$A$ は $X$ において最小元 $m$ を持つ：
  $$
  m \in A,\quad \forall a \in A,\ m \leq a
  $$
- $A \subset Y$ より $m \in Y$ であり、$Y$ 上の順序は $X$ の順序の制限なので、$m$ は $A$ の（$Y$ における）最小元でもある。
- よって、$(Y,\leq)$ の任意の空でない部分集合は最小元を持つ。
- 以上より $(Y,\leq)$ は整列集合である。

__(2) 超限帰納法を用いる方針（発展）__

- $X$ 上の超限帰納法を用いて、$Y$ の整列性を示すことを考える。
- 命題 $P(x)$ を次のように設定する：
  $$
  P(x): \quad \text{「}Y \cap X_{\le x} = \{y \in Y \mid y \le x\}\text{ は整列集合である」}
  $$
  - ここで $X_{\le x} = \{y \in X \mid y \le x\}$ は $x$ 以下の元全体。
- 帰納法のステップ：
  - 基底：最小元 $m \in X$ について、$Y \cap X_{\le m}$ は高々1点（$m$ が $Y$ に属するかどうかで変わる）であり、明らかに整列集合。
  - 帰納段階：任意の $x \in X$ について、「すべての $y < x$ で $P(y)$ が成り立つ」と仮定し、$P(x)$ を示す。
    - $Y \cap X_{\le x}$ は、$Y \cap X_{<x}$ と（もし $x \in Y$ なら）一点 $\{x\}$ の和集合として書ける。
    - 帰納仮定より $Y \cap X_{<x}$ は整列集合であり、一点集合も整列集合なので、それらの和集合（適切に順序を入れたもの）も整列集合であることを示す（通常、整列集合に最大元を1つ加えたものは再び整列集合）。
- すべての $x \in X$ で $P(x)$ が成り立てば、特に $x$ を $X$ の最大元（存在しない場合は上限を考える）とすることで $Y$ 全体が整列集合であることが従う。

__問6（超限帰納法の正当性）__

超限帰納法が「すべての $x \in X$ で $P(x)$ が成り立つ」ことを保証する理由を、背理法で説明する。

- 整列集合 $(X,\leq)$ と命題 $P(x)$ に対し、超限帰納法の条件：
  > 任意の $x \in X$ について、  
  > 「すべての $y < x$ に対して $P(y)$ が成り立つならば $P(x)$ も成り立つ」
  が成り立っているとする。
- ここで、ある $x \in X$ で $P(x)$ が偽になると仮定する。
- $A = \{x \in X \mid \lnot P(x)\}$ とおくと、$A$ は空でない。
- $(X,\leq)$ は整列集合なので、$A$ には最小元 $m$ が存在する。
- $m$ が $A$ の最小元であることから、すべての $y < m$ について $P(y)$ は真（もし偽なら $y \in A$ となり $m$ の最小性に反する）。
- したがって、「すべての $y < m$ に対して $P(y)$ が成り立つ」が真である。
- 超限帰納法の条件より、このとき $P(m)$ も成り立たなければならない。
- しかし $m \in A$ より $P(m)$ は偽 → 矛盾。
- よって、そのような $x$ は存在せず、すべての $x \in X$ で $P(x)$ が真である。

__問7（順序数の加法の結合則）__

命題：
> すべての順序数 $\alpha,\beta,\gamma$ について、
$$
(\alpha + \beta) + \gamma = \alpha + (\beta + \gamma)
$$
> が成り立つ。

__帰納の変数__

- $\alpha,\beta$ を固定し、**$\gamma$ に関する超限帰納法**を用いる。

__3段階の内容__

**1. 基底（$\gamma = 0$）**

- 示すべきこと：
  $$
  (\alpha + \beta) + 0 = \alpha + (\beta + 0)
  $$
- 順序数の加法の定義（$\xi + 0 = \xi$）より、
  - 左辺：$(\alpha + \beta) + 0 = \alpha + \beta$
  - 右辺：$\alpha + (\beta + 0) = \alpha + \beta$
- よって基底では等式が成立。

**2. 後続段階（$\gamma = \delta + 1$）** 

- 帰納仮定：$\delta$ については
  $$
  (\alpha + \beta) + \delta = \alpha + (\beta + \delta)
  $$
  が成り立つと仮定する。
- 示すべきこと：$\gamma = \delta + 1$ について
  $$
  (\alpha + \beta) + (\delta + 1) = \alpha + (\beta + (\delta + 1))
  $$
- 順序数の加法の定義（$\xi + (\eta + 1) = (\xi + \eta) + 1$）を用いて変形する：
  - 左辺：$(\alpha + \beta) + (\delta + 1) = ((\alpha + \beta) + \delta) + 1$
  - 右辺：$\alpha + (\beta + (\delta + 1)) = \alpha + ((\beta + \delta) + 1) = (\alpha + (\beta + \delta)) + 1$
- 帰納仮定より $(\alpha + \beta) + \delta = \alpha + (\beta + \delta)$ なので、両辺に $+1$ を施しても等しい。
- よって後続段階でも等式が成立。

**3. 極限段階（$\gamma$ が極限順序数）**

- $\gamma$ を極限順序数とする。
- 帰納仮定：すべての $\delta < \gamma$ について
  $$
  (\alpha + \beta) + \delta = \alpha + (\beta + \delta)
  $$
  が成り立つと仮定する。
- 示すべきこと：
  $$
  (\alpha + \beta) + \gamma = \alpha + (\beta + \gamma)
  $$
- 順序数の加法の定義（極限順序数に対する和は「それより小さい順序数での和の上限」）より：
  - 左辺： $(\alpha + \beta) + \gamma = \sup_{\delta < \gamma} ((\alpha + \beta) + \delta)$
  - 右辺： $\alpha + (\beta + \gamma) = \sup_{\delta < \gamma} (\alpha + (\beta + \delta))$
- 帰納仮定より、各 $\delta < \gamma$ について $(\alpha + \beta) + \delta = \alpha + (\beta + \delta)$ なので、両者の上限も等しい。
- よって極限段階でも等式が成立。

以上より、$\gamma$ に関する超限帰納法により、すべての $\gamma$ について結合則が成り立つ。  
（$\alpha,\beta$ は任意だったので、すべての $\alpha,\beta,\gamma$ について成立。）
