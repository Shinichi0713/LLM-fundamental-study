# 添数づけられた族と選択公理

## 添数づけられた族

「添数づけられた族（indexed family）」は、集合論・数学でよく使われる概念で、**集合の集まり（族）に、添数（インデックス）を付けて扱うための仕組み**です。

### 1. 直感的なイメージ

- 普通の「集合の族」は、単に「集合をいくつか集めたもの」です。
  - 例：{A, B, C} は3つの集合 A, B, C からなる族。
- 一方、「添数づけられた族」は、**各集合にラベル（添数）を付けて**、  
  「どの集合がどの添数に対応するか」を明示したものです。
  - 例：A₁, A₂, A₃ のように、1, 2, 3 という添数で集合を区別する。

この「添数」は、自然数とは限らず、一般の集合（インデックス集合）から取れます。

### 2. 形式的な定義

__インデックス集合（添字集合）__

まず、**添数全体の集合**を用意します。これを**インデックス集合（index set）** と呼び、通常 I などで表します。

- 例：I = {1, 2, 3} や I = ℕ（自然数全体）など。

__添数づけられた族の定義__

**インデックス集合 I で添数づけられた集合の族**とは、  
「I の各元 i に対して、ある集合 Aᵢ を対応させる写像」のことです。

より正確には：

- ある集合 X を全体集合とする。
- インデックス集合 I がある。
- 写像  
  $$
  f \colon I \to \mathcal{P}(X)
  $$  
  が与えられる（ここで $\mathcal{P}(X)$ は X のべき集合）。
- このとき、各 i ∈ I に対して  
  $$
  A_i := f(i)
  $$  
  とおくと、集合の集まり  
  $$
  \{A_i\}_{i \in I}
  $$  
  を **I で添数づけられた集合の族**と呼ぶ。

記号としては、

- $\{A_i\}_{i \in I}$
- $(A_i)_{i \in I}$

などと書きます。

### 3. 具体例

__例1：有限インデックス集合__

- I = {1, 2, 3}
- A₁ = {1, 2}, A₂ = {3, 4}, A₃ = {5, 6}

このとき、  
$\{A_i\}_{i \in I}$ は、添数 1, 2, 3 でラベルされた3つの集合の族です。

__例2：自然数を添数とする族__

- I = ℕ（自然数全体）
- Aₙ = {n, n+1} と定義する。

このとき、  
$\{A_n\}_{n \in \mathbb{N}}$ は  
A₁ = {1, 2}, A₂ = {2, 3}, A₃ = {3, 4}, … という無限個の集合の族です。

__例3：実数を添数に使う__

- I = ℝ（実数全体）
- 各 r ∈ ℝ に対して、Aᵣ = {x ∈ ℝ | x ≥ r} と定義する。

このとき、  
$\{A_r\}_{r \in \mathbb{R}}$ は、実数 r を添数とする「半直線」の族です。

### 4. 添数づけられた族を使うメリット

1. **同じ集合が複数回現れても区別できる**  
   - 普通の集合の族では、同じ集合が2回現れても1つとみなされますが、  
     添数づけられた族では、**添数が違えば別の元**とみなせます。
   - 例：A₁ = {1, 2}, A₂ = {1, 2} でも、添数が違うので区別されます。

2. **無限族を扱いやすい**  
   - 自然数 ℕ や実数 ℝ など、無限集合をインデックス集合にすることで、  
     無限個の集合をシステマティックに扱えます。

3. **和集合・共通部分・直積などを定義しやすい**  
   - 添数づけられた族 $\{A_i\}_{i \in I}$ に対して、
     - 和集合：$\bigcup_{i \in I} A_i$
     - 共通部分：$\bigcap_{i \in I} A_i$
     - 直積：$\prod_{i \in I} A_i$  
     などが自然に定義できます。

## 集合族の和

「集合族の和」には、大きく分けて2つの意味があります。

1. **有限個の集合の和（普通の和集合）**
2. **一般の集合族（添数づけられた族を含む）の和**

順に説明します。

### 1. 有限個の集合の和（普通の和集合）

__定義（2つの集合の場合）__

2つの集合 A, B に対して、**和集合（union）** $A \cup B$ は、

$$
A \cup B = \{x \mid x \in A \text{ または } x \in B\}
$$

と定義されます。  
つまり、「A か B の少なくとも一方に属する元全体」です。

__3つ以上の有限個の集合の場合__

同様に、有限個の集合 A₁, A₂, …, Aₙ に対して、

$$
A_1 \cup A_2 \cup \dots \cup A_n = \{x \mid \text{ある } i \in \{1,\dots,n\} \text{ について } x \in A_i\}
$$

と定義します。

### 2. 一般の集合族の和

ここからが本題です。  
「集合族」とは、**集合を要素とする集まり**のことです。

__2.1 集合族の定義（形式的）__

集合 X を全体集合とし、その部分集合の集まり $\mathcal{F}$ を考えます。  
$\mathcal{F}$ が**集合族**であるとは、

- $\mathcal{F} \subset \mathcal{P}(X)$  
  （$\mathcal{P}(X)$ は X のべき集合）

という意味です。

例：
- $\mathcal{F} = \{A, B, C\}$  
  ここで A, B, C は X の部分集合。

__2.2 集合族の和の定義__

集合族 $\mathcal{F}$ に対して、その**和集合**（union）は

$$
\bigcup \mathcal{F} = \{x \in X \mid \text{ある } A \in \mathcal{F} \text{ について } x \in A\}
$$

と定義されます。

言葉で言うと：

> 「族 $\mathcal{F}$ に属する少なくとも1つの集合に含まれる元全体」

です。

__2.3 添数づけられた族の場合__

集合族が**添数づけられた族** $\{A_i\}_{i \in I}$ の形で与えられていることも多いです。  
この場合、和集合は

$$
\bigcup_{i \in I} A_i = \{x \mid \text{ある } i \in I \text{ について } x \in A_i\}
$$

と書かれます。

これは、先ほどの定義

$$
\bigcup \mathcal{F} \quad\text{（ただし } \mathcal{F} = \{A_i \mid i \in I\}\text{）}
$$

と同じ意味です。

### 3. 具体例

__例1：有限族__

- X = ℝ（実数全体）
- A = [0, 1], B = [1, 2], C = [2, 3]
- $\mathcal{F} = \{A, B, C\}$

このとき、

$$
\bigcup \mathcal{F} = A \cup B \cup C = [0, 3]
$$

です。

__例2：自然数を添数とする無限族__

- I = ℕ（自然数全体）
- 各 n ∈ ℕ に対して、Aₙ = [n, n+1]（閉区間）
- 族 $\{A_n\}_{n \in \mathbb{N}}$

このとき、

$$
\bigcup_{n \in \mathbb{N}} A_n = [1, \infty)
$$

となります（1以上の実数全体）。

__例3：実数を添数とする族__

- I = ℝ
- 各 r ∈ ℝ に対して、Aᵣ = {x ∈ ℝ | x ≥ r}
- 族 $\{A_r\}_{r \in \mathbb{R}}$

このとき、

$$
\bigcup_{r \in \mathbb{R}} A_r = \mathbb{R}
$$

です（任意の実数 x に対して、r ≤ x となる r を取れば x ∈ Aᵣ となるため）。

### 4. まとめ

- **有限個の集合の和**：  
  $A_1 \cup A_2 \cup \dots \cup A_n = \{x \mid \text{ある } i \text{ について } x \in A_i\}$
- **一般の集合族 $\mathcal{F}$ の和**：  
  $\bigcup \mathcal{F} = \{x \mid \text{ある } A \in \mathcal{F} \text{ について } x \in A\}$
- **添数づけられた族 $\{A_i\}_{i \in I}$ の和**：  
  $\bigcup_{i \in I} A_i = \{x \mid \text{ある } i \in I \text{ について } x \in A_i\}$

これらはすべて、「少なくとも1つの集合に属する元全体」という同じ考え方を、有限・無限・一般の族に対して統一的に表現したものです。
