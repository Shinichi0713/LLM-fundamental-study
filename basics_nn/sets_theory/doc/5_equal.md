# 集合の対等

## 対等

集合論における**対等（equinumerous）** とは、**2つの集合の要素の個数が「同じ」であること**を厳密に表す概念です。  
有限集合なら「要素数が等しい」ことに対応し、無限集合にも自然に拡張できます。


### 1. 対等の定義

2つの集合 $A, B$ が**対等**であるとは、$A$ から $B$ への**全単射（bijection）** が存在することをいいます。

記号では、
$$
A \sim B
$$
と書くことがあります（ここでの $\sim$ は同値関係の記号とは別物ですが、文脈で区別します）。

より正確に書くと：

- 写像 $f: A \to B$ が存在して、
  - **単射**：$x \neq y \implies f(x) \neq f(y)$
  - **全射**：任意の $b \in B$ に対し、ある $a \in A$ が存在して $f(a) = b$

このような $f$ が存在するとき、$A$ と $B$ は**対等**であるといいます。


### 2. 具体例

__例1：有限集合__
- $A = \{1,2,3\},\quad B = \{a,b,c\}$
- $f(1)=a,\ f(2)=b,\ f(3)=c$ は全単射
- よって $A$ と $B$ は対等（どちらも要素数 3）

__例2：無限集合__
- $A = \mathbb{N} = \{1,2,3,\dots\},\quad B = \{2,4,6,\dots\}$（正の偶数の集合）
- $f(n) = 2n$ は $\mathbb{N} \to B$ の全単射
- よって $\mathbb{N}$ と $B$ は対等（どちらも可算無限）

__例3：対等でない例__
- $A = \{1,2\},\quad B = \{a,b,c\}$
- $A$ から $B$ への全単射は存在しない（要素数が異なる）
- よって $A$ と $B$ は対等ではない

### 3. 対等と濃度（cardinality）
この部分は後ほど詳細に扱っていきます。
関連する内容ということでご承知おきください。

- 対等である集合は、**同じ濃度（cardinality）を持つ**といいます。
- 有限集合の場合、濃度は単に「要素の個数」です。
- 無限集合の場合も、対等であるかどうかで「大きさが同じか」を判断します（例：自然数全体と有理数全体は対等、実数全体はそれらより「大きい」）。


### 4. 注意点

- 「対等」は**集合のサイズ（濃度）が同じ**という意味であり、集合が「等しい」とは別概念です。
  - 例：$\{1,2,3\}$ と $\{a,b,c\}$ は等しくないが、対等である。
- 対等関係は、集合全体のクラス上の**同値関係**になります（反射律・対称律・推移律を満たす）。

## 濃度

**濃度（cardinality）** とは、集合の「大きさ」や「要素の個数」を表す数学的な概念です。  
有限集合では単に「要素の数」に対応し、無限集合にも自然に拡張されます。


### 1. 濃度の直感的な意味

- 有限集合の場合：  
  集合 $A$ の濃度は、その中に含まれる要素の個数です。  
  例：$A = \{1,2,3\}$ の濃度は 3、記号で $|A| = 3$ などと書きます。

- 無限集合の場合：  
  「要素の数」を直接数えることはできませんが、**他の集合との対等関係（全単射の有無）** を通じて「大きさが同じか・どちらが大きいか」を比較します。


### 2. 濃度の厳密な定義（対等による）

2つの集合 $A, B$ が**対等（equinumerous）** であるとは、$A$ から $B$ への**全単射**が存在することをいいます（前の回答で説明した通り）。

このとき、$A$ と $B$ は**同じ濃度を持つ**と定義します。

- 有限集合の場合：  
  要素数が等しければ全単射が作れるので、「濃度が同じ」＝「要素数が同じ」。

- 無限集合の場合：  
  例：自然数全体 $\mathbb{N}$ と偶数全体は全単射 $n \mapsto 2n$ で結べるので、同じ濃度（可算無限）です。


### 3. 濃度の記号と例

- 有限集合の濃度：自然数 $0,1,2,\dots$ で表す。  
  例：$|\{a,b,c\}| = 3$

- 可算無限濃度：$\aleph_0$（アレフ・ゼロ）  
  - 自然数全体 $\mathbb{N}$、整数全体 $\mathbb{Z}$、有理数全体 $\mathbb{Q}$ などがこの濃度を持つ。

- 連続体濃度：$\mathfrak{c}$  
  - 実数全体 $\mathbb{R}$、区間 $(0,1)$ などがこの濃度を持つ。  
  - 可算無限より「大きい」ことが知られています（カントールの対角線論法）。

### 4. 濃度の比較

集合 $A, B$ について：

- $|A| = |B|$：$A$ と $B$ の間に全単射が存在する（対等）。
- $|A| \leq |B|$：$A$ から $B$ への単射が存在する（$A$ は $B$ より大きくない）。
- $|A| < |B|$：$|A| \leq |B|$ だが $|A| \neq |B|$（$B$ の方が真に大きい）。

このように、濃度は**全単射・単射の存在**を通じて比較されます。


### 5. なぜ濃度が必要か

- 有限集合では「個数」で十分ですが、無限集合では「個数を数える」ことができません。
- そこで、**全単射の有無**を「大きさが同じか」の基準にすることで、無限集合のサイズを厳密に比較できます。
- これにより、「自然数と有理数は同じくらい多いが、実数はもっと多い」といったことが数学的に証明できます。


## 濃度の大小

**濃度の大小**とは、集合の「大きさ」を比較するための数学的な関係です。  
有限集合では単に「要素数が多い／少ない」に対応し、無限集合にも自然に拡張されます。

### 1. 濃度の大小の定義

2つの集合 $A, B$ について、濃度 $|A|, |B|$ の大小は次のように定義されます。

__(1) $|A| \leq |B|$__
- 定義：$A$ から $B$ への**単射（injection）** が存在するとき、$|A| \leq |B|$ と書く。
- 意味：$A$ の要素を $B$ の要素に「重複なく」対応づけられる  
  → $A$ の「大きさ」は $B$ より大きくない（$B$ の方が同じかそれ以上）。

__(2) $|A| = |B|$__  
- 定義：$A$ から $B$ への**全単射（bijection）** が存在するとき、$|A| = |B|$ と書く。
- 意味：$A$ と $B$ は**対等**（equinumerous）で、同じ濃度を持つ。

__(3) $|A| < |B|$__
- 定義：$|A| \leq |B|$ だが $|A| \neq |B|$ のとき、$|A| < |B|$ と書く。
- 意味：$A$ から $B$ への単射はあるが、全単射は存在しない  
  → $B$ の方が**真に大きい**濃度を持つ。

### 2. 具体例

__例1：有限集合__
- $A = \{1,2\},\quad B = \{a,b,c\}$
- $A \to B$ の単射は存在する（例：$1 \mapsto a,\ 2 \mapsto b$）
- しかし全単射は存在しない（要素数が異なる）
- よって $|A| < |B|$

__例2：無限集合（可算無限と連続体濃度）__
- $|\mathbb{N}| = \aleph_0$（可算無限）
- $|\mathbb{R}| = \mathfrak{c}$（連続体濃度）
- $\mathbb{N} \to \mathbb{R}$ の単射は明らかに存在する（自然数をそのまま実数とみなす）
- しかしカントールの対角線論法により、$\mathbb{N} \to \mathbb{R}$ の全単射は存在しないことが知られている
- よって $|\mathbb{N}| < |\mathbb{R}|$

__例3：同じ無限濃度__
- $|\mathbb{N}| = |\mathbb{Z}| = |\mathbb{Q}| = \aleph_0$  
  （自然数・整数・有理数はすべて可算無限で、互いに全単射が構成できる）

### 3. 濃度の大小が持つ数学的意味

__(1) 「大きさ」の厳密な比較__
- 有限集合では「個数」で比較できますが、無限集合では「個数を数える」ことができません。
- そこで、**単射・全単射の存在**を「大きさが同じか・どちらが大きいか」の基準にすることで、無限集合のサイズを厳密に比較できます。

__(2) 無限の階層の存在を示す__
- 自然数全体 $\mathbb{N}$ と実数全体 $\mathbb{R}$ は、どちらも無限ですが、$\mathbb{R}$ の方が「真に大きい」ことが証明できます。
- さらに、任意の集合 $X$ に対して、そのべき集合 $\mathcal{P}(X)$ は $|X| < |\mathcal{P}(X)|$ を満たします（カントールの定理）。  
  これにより、**無限に大きい濃度の階層**が存在することがわかります。

__(3) 同値関係・順序関係としての構造__
- 「$|A| = |B|$」は対等関係であり、反射律・対称律・推移律を満たす**同値関係**です。
- 「$|A| \leq |B|$」は反射的・推移的であり、濃度全体のクラス上の**前順序（preorder）** になります（反対称性も成り立ち、実際には**全順序的な性質**を持ちますが、選択公理を仮定すると整列可能定理により「すべての濃度は比較可能」とみなせます）。

### 4. なぜこの定義なのか（直感との一致）

- 有限集合の場合：  
  $|A| \leq |B|$ は「$A$ の要素数 ≤ $B$ の要素数」と完全に一致します。
- 無限集合の場合：  
  「$A$ を $B$ に埋め込める（単射がある）」なら、$A$ は $B$ より大きくない、という直感を数学的に表現したものです。

この定義により、**有限・無限を問わず一貫した「大きさ」の比較**が可能になります。

### 5. Cantor-Bersteinの定理

**Cantor–Bernsteinの定理**（シュレーダー–ベルンシュタインの定理とも呼ばれます）は、**濃度の比較に関する基本的で強力な定理**です。

__1. 定理の主張__

2つの集合 $A, B$ について、

- $A$ から $B$ への**単射**が存在し、
- $B$ から $A$ への**単射**が存在する

ならば、$A$ から $B$ への**全単射**が存在する、すなわち
$$
|A| = |B|
$$
が成り立つ、というのが定理の主張です。

記号で書くと：
$$
|A| \leq |B| \quad\text{かつ}\quad |B| \leq |A| \implies |A| = |B|
$$

__2. 直感的な意味__

- $|A| \leq |B|$：$A$ を $B$ に「埋め込める」 → $A$ は $B$ より大きくない
- $|B| \leq |A|$：$B$ を $A$ に「埋め込める」 → $B$ は $A$ より大きくない

この2つが同時に成り立つなら、**どちらかが真に大きいということはあり得ず、結局は同じ大きさ（同じ濃度）になる**、というのが定理の内容です。

有限集合では当たり前に感じられますが、無限集合でも成り立つことが重要です。

__3. 具体例__

__例：開区間 $(0,1)$ と閉区間 $[0,1]$__

- $(0,1) \subset [0,1]$ なので、包含写像により単射 $(0,1) \to [0,1]$ が存在  
  → $|(0,1)| \leq |[0,1]|$
- 一方、$[0,1] \to (0,1)$ の単射も構成できます（例：$x \mapsto \frac{x}{3} + \frac{1}{3}$）  
  → $|[0,1]| \leq |(0,1)|$

Cantor–Bernsteinの定理より、
$$
|(0,1)| = |[0,1]|
$$
つまり、開区間と閉区間は**同じ濃度（連続体濃度）** を持つことがわかります。

__4. なぜ重要なのか__

1. **濃度の順序関係が「反対称的」になる**  
   $|A| \leq |B|$ かつ $|B| \leq |A|$ なら $|A| = |B|$ が成り立つので、濃度の比較は**半順序**（実際には全順序的な性質）を持ちます。

2. **全単射を直接構成しなくても濃度が等しいとわかる**  
   単射を2つ見つけるだけで「同じ濃度」と結論できるため、**全単射を具体的に構成する手間を省ける**ことが多いです。

3. **無限集合のサイズ比較の基礎**  
   可算無限・連続体濃度など、さまざまな無限の大きさを比較する際の基本的な道具になります。

__5. 証明__

以下、Cantor–Bernsteinの定理の証明を示します。

---

__証明の準備__

- 単射 $f: A \to B$、単射 $g: B \to A$ が与えられているとします。
- 目標：全単射 $h: A \to B$ を構成する。

証明のアイデアは、**$A$ を2つの部分に分け、片方では $f$ を、もう片方では $g^{-1}$ を使う**ことです。  
ただし $g$ は単射であっても全射とは限らないので、「逆写像」はそのままでは定義できません。そこで、**逆像**を使って慎重に扱います。

__証明のステップ__

__ステップ1：写像 $F: \mathcal{P}(A) \to \mathcal{P}(A)$ の定義__

$A$ の部分集合 $X \subset A$ に対して、
$$
F(X) = A \setminus g(B \setminus f(X))
$$
と定義します。ここで、

- $f(X) = \{f(x) \mid x \in X\} \subset B$
- $B \setminus f(X)$：$B$ のうち $f(X)$ に含まれない部分
- $g(B \setminus f(X)) = \{g(y) \mid y \in B \setminus f(X)\} \subset A$
- $A \setminus g(B \setminus f(X))$：$A$ のうち、$g(B \setminus f(X))$ に含まれない部分

つまり、$F(X)$ は「$f(X)$ に写されなかった $B$ の元を $g$ で $A$ に戻したもの」の**補集合**です。

この $F$ は**単調増加**です：
$$
X \subset Y \implies F(X) \subset F(Y)
$$
（証明略：包含関係を追えば確認できます）

__ステップ2：不動点の存在__

単調増加写像 $F: \mathcal{P}(A) \to \mathcal{P}(A)$ に対し、次の集合を考えます：
$$
C = \bigcup \{X \subset A \mid X \subset F(X)\}
$$
（「$X \subset F(X)$ を満たすすべての $X$ の和集合」）

このとき、次が成り立ちます：

1. $C \subset F(C)$
2. $F(C) \subset C$

したがって、$F(C) = C$ となり、$C$ は $F$ の**不動点**です。

（証明の概略：単調性と和集合の定義から、包含関係を両方向に示せます。詳細は集合論の教科書を参照してください。）

__ステップ3：全単射 $h: A \to B$ の構成__

不動点 $C \subset A$ に対して、次のように $h: A \to B$ を定義します：
$$
h(x) = 
\begin{cases}
f(x) & \text{if } x \in C \\
g^{-1}(x) & \text{if } x \in A \setminus C
\end{cases}
$$

ここで、$g^{-1}(x)$ は「$g(y) = x$ となる $y \in B$」の意味です。  
$g$ は単射なので、そのような $y$ は高々1つですが、存在するかどうかが問題です。  
実は、$x \in A \setminus C$ のとき、必ずそのような $y$ が存在することが、不動点の条件から導かれます。

より正確に：

- $C = F(C) = A \setminus g(B \setminus f(C))$
- よって $A \setminus C = g(B \setminus f(C))$
- したがって、任意の $x \in A \setminus C$ に対し、ある $y \in B \setminus f(C)$ が一意に存在して $g(y) = x$
- この $y$ を $g^{-1}(x)$ と書く

この定義により、$h$ は well-defined です。

__ステップ4：$h$ が全単射であることの確認__

**(a) 単射であること**

- $x_1, x_2 \in A$ で $h(x_1) = h(x_2)$ と仮定します。
- $x_1, x_2$ がともに $C$ に属する場合：  
  $h(x_1) = f(x_1),\ h(x_2) = f(x_2)$ で、$f$ は単射なので $x_1 = x_2$。
- $x_1, x_2$ がともに $A \setminus C$ に属する場合：  
  $h(x_1) = g^{-1}(x_1),\ h(x_2) = g^{-1}(x_2)$ で、$g$ は単射なので $g^{-1}$ も単射的 → $x_1 = x_2$。
- 一方が $C$、他方が $A \setminus C$ に属する場合は起こり得ません。  
  なぜなら、$h(C) = f(C)$、$h(A \setminus C) = g^{-1}(A \setminus C) \subset B \setminus f(C)$ であり、  
  $f(C) \cap (B \setminus f(C)) = \varnothing$ だからです。

よって $h$ は単射。

**(b) 全射であること**

任意の $y \in B$ を取ります。

- もし $y \in f(C)$ なら、ある $x \in C$ が存在して $f(x) = y$。このとき $h(x) = f(x) = y$。
- もし $y \in B \setminus f(C)$ なら、$g(y) \in g(B \setminus f(C)) = A \setminus C$。  
  このとき $h(g(y)) = g^{-1}(g(y)) = y$。

したがって、任意の $y \in B$ に対して $h(x) = y$ となる $x \in A$ が存在するので、$h$ は全射。

以上より、$h: A \to B$ は全単射です。

__結論__

単射 $f: A \to B$ と単射 $g: B \to A$ が存在するとき、上記の構成により全単射 $h: A \to B$ が得られる。  
したがって、
$$
|A| \leq |B| \quad\text{かつ}\quad |B| \leq |A| \implies |A| = |B|
$$
が成り立ち、Cantor–Bernsteinの定理が証明されました。

### 6. Cantorの定理

**Cantorの定理**は、**任意の集合に対して、そのべき集合（部分集合全体の集合）の濃度は、元の集合の濃度より真に大きい**ことを主張する定理です。  
無限集合であっても「もっと大きい無限」が存在することを示す、集合論の基本的で重要な結果です。

__1. 定理の主張__

集合 $X$ の**べき集合（power set）** を
$$
\mathcal{P}(X) = \{A \mid A \subset X\}
$$
と書きます（$X$ の部分集合全体の集合）。

このとき、Cantorの定理は次のように述べられます：

> 任意の集合 $X$ に対し、  
$$
|X| < |\mathcal{P}(X)|
$$
> が成り立つ。  
> すなわち、$X$ から $\mathcal{P}(X)$ への**全射は存在しない**。

特に、$X$ が無限集合であっても、$\mathcal{P}(X)$ はそれより真に大きい濃度を持ちます。

__2. 直感的な意味__

- 有限集合の場合：  
  $|X| = n$ なら $|\mathcal{P}(X)| = 2^n$ であり、$n < 2^n$ は明らかです。
- 無限集合の場合：  
  例えば $X = \mathbb{N}$（自然数全体）とすると、$\mathcal{P}(\mathbb{N})$ は $\mathbb{N}$ より真に大きい濃度（連続体濃度以上）を持ちます。  
  つまり、「自然数の部分集合全体」は自然数全体より「はるかに多い」ということです。

この定理により、**無限にも大小の階層があり、いくらでも大きい無限が存在する**ことがわかります。



__3. 証明の概要（対角線論法）__

---

Cantorの定理の証明は、**対角線論法（diagonal argument）** を用います。

__ステップ1：全射が存在しないことを示す__

目標：$X$ から $\mathcal{P}(X)$ への**全射は存在しない**ことを示す。  
（全射がなければ、全単射も存在せず、したがって $|X| < |\mathcal{P}(X)|$）

背理法で示します。

- 仮定：ある全射 $f: X \to \mathcal{P}(X)$ が存在すると仮定する。
- このとき、各 $x \in X$ に対して $f(x) \subset X$ が定まる。

__ステップ2：対角集合の構成__

次の集合 $D \subset X$ を考えます：
$$
D = \{x \in X \mid x \notin f(x)\}
$$
つまり、$D$ は「自分自身を要素として含まない $f(x)$ に対応する $x$ 全体」です。

- $D$ は $X$ の部分集合なので、$D \in \mathcal{P}(X)$。
- $f$ は全射なので、ある $d \in X$ が存在して $f(d) = D$ となるはずです。

__ステップ3：矛盾の導出__

この $d$ について、次の2通りを考えます。

1. $d \in D$ の場合  
   - $D$ の定義より、$d \in D \implies d \notin f(d)$。
   - しかし $f(d) = D$ なので、$d \notin D$ となり矛盾。

2. $d \notin D$ の場合  
   - $D$ の定義より、$d \notin D \implies d \in f(d)$。
   - しかし $f(d) = D$ なので、$d \in D$ となり矛盾。

どちらの場合も矛盾が生じます。  
したがって、最初の仮定「全射 $f: X \to \mathcal{P}(X)$ が存在する」は誤りです。

__ステップ4：濃度の比較__

- 包含写像 $x \mapsto \{x\}$ は $X \to \mathcal{P}(X)$ の単射なので、$|X| \leq |\mathcal{P}(X)|$。
- 一方、全射は存在しないので、全単射も存在せず、$|X| \neq |\mathcal{P}(X)|$。

よって、
$$
|X| < |\mathcal{P}(X)|
$$
が成り立ちます。

---

__4. 定理の意義__

1. **無限の階層の存在**  
   - 自然数全体 $\mathbb{N}$ に対して、$\mathcal{P}(\mathbb{N})$ はより大きい濃度を持つ。
   - さらに $\mathcal{P}(\mathcal{P}(\mathbb{N}))$ はそれより大きく、この操作を繰り返すことで**無限に大きい濃度の列**が得られる。

2. **連続体仮説との関係**  
   - $|\mathbb{N}| = \aleph_0$、$|\mathcal{P}(\mathbb{N})| = 2^{\aleph_0}$ であり、これは実数全体 $\mathbb{R}$ の濃度（連続体濃度）に等しいことが知られています。
   - 連続体仮説は「$2^{\aleph_0} = \aleph_1$ か？」という問いであり、Cantorの定理はその出発点となります。

3. **数学の基礎への影響**  
   - 「すべての集合の濃度は比較可能か？」「より大きい無限が常に存在するか？」といった問題は、集合論の公理（ZFC）や選択公理・連続体仮説と深く結びついています。

## 有限集合、無限集合

**有限集合**と**無限集合**は、集合の「大きさ」を区別する基本的な概念です。  
数学的には、**自然数との全単射の有無**によって定義されます。

### 1. 有限集合の定義

集合 $X$ が**有限集合**であるとは、ある自然数 $n \in \mathbb{N}$ に対して、
$$
X \sim \{1,2,\dots,n\}
$$
が成り立つことをいいます。ここで $\sim$ は**対等**（全単射が存在する）を表します。

言い換えると：

- $X$ から $\{1,2,\dots,n\}$ への**全単射**が存在する
- このとき、$X$ の**濃度（要素数）** は $n$ であるといい、$|X| = n$ と書く

例：
- $X = \{a,b,c\}$ は $\{1,2,3\}$ と全単射があるので有限集合、$|X| = 3$
- 空集合 $\emptyset$ は $n=0$ に対応し、有限集合とみなす（$|\emptyset| = 0$）

### 2. 無限集合の定義

集合 $X$ が**無限集合**であるとは、**有限集合でない**こと、すなわちどの自然数 $n$ に対しても
$$
X \not\sim \{1,2,\dots,n\}
$$
が成り立つことをいいます。

より実用的な同値定義として：

- $X$ が無限集合  
  $\iff$ $X$ から自身への**単射であって全射でない写像**が存在する  
  $\iff$ $X$ はある真部分集合と対等（同じ濃度を持つ）

例：
- 自然数全体 $\mathbb{N}$：写像 $n \mapsto n+1$ は単射だが全射でない → 無限集合
- 実数全体 $\mathbb{R}$：同様に無限集合

### 3. 可算無限と非可算無限

無限集合の中でも特に重要な区別があります。

__(1) 可算無限集合（countably infinite）__
- 自然数全体 $\mathbb{N}$ と対等な無限集合
- 例：$\mathbb{Z}$（整数全体）、$\mathbb{Q}$（有理数全体）など
- 濃度は $\aleph_0$（アレフ・ゼロ）と書く

__(2) 非可算無限集合（uncountably infinite）__
- $\mathbb{N}$ と対等でない無限集合
- 例：$\mathbb{R}$（実数全体）、$\mathcal{P}(\mathbb{N})$（自然数のべき集合）など
- 実数全体の濃度は**連続体濃度** $\mathfrak{c}$ と呼ばれる

### 4. 有限集合と無限集合の性質の違い

__(1) 部分集合との関係__
- 有限集合：真部分集合は必ず**より小さい濃度**を持つ  
  （例：$\{1,2,3\}$ の真部分集合の濃度は 0,1,2）
- 無限集合：真部分集合であっても**同じ濃度**を持つことがある  
  （例：$\mathbb{N}$ と偶数全体は同じ濃度 $\aleph_0$）

__(2) 和集合・直積の振る舞い__
- 有限集合の有限和・有限直積はまた有限集合
- 無限集合との和集合・直積は多くの場合無限集合（特に可算無限＋有限＝可算無限 など）

__(3) 選択公理との関係__
- 有限集合の族から元を選ぶのは自明
- 無限集合の族から同時に元を選ぶには**選択公理**が必要になる場合が多い

## 可算集合、非可算集合

**可算集合**と**非可算集合**は、無限集合を「どれくらい大きいか」で分類する概念です。  
数学的には、**自然数全体 $\mathbb{N}$ との全単射の有無**によって定義されます。

### 1. 可算集合（countable set）

__定義__

集合 $X$ が**可算集合**であるとは、次のいずれか（同値）を満たすことをいいます。

1. $X$ が有限集合である
2. $X$ が自然数全体 $\mathbb{N}$ と対等である（全単射が存在する）

特に 2 のケースを**可算無限集合**と呼びます。

- 記号：可算無限集合の濃度を $\aleph_0$（アレフ・ゼロ）と書く
- 意味：要素を $a_1, a_2, a_3, \dots$ と「番号付け」できる

__具体例__

- $\mathbb{N} = \{1,2,3,\dots\}$：自明に可算無限（恒等写像が全単射）
- $\mathbb{Z} = \{\dots,-2,-1,0,1,2,\dots\}$（整数全体）  
  全単射の例：$f(n) = \begin{cases} k & n=2k \\ -k & n=2k+1 \end{cases}$ など → 可算無限
- $\mathbb{Q}$（有理数全体）  
  - 有理数は既約分数 $p/q$ で表せるので、格子点 $(p,q)$ と対応づけられる  
  - 格子点は自然数と全単射が作れる → 可算無限

### 2. 非可算集合（uncountable set）

__定義__

集合 $X$ が**非可算集合**であるとは、**可算集合でない**こと、すなわち

- $X$ は有限集合ではなく、
- $X$ は $\mathbb{N}$ と対等でもない

ことをいいます。

言い換えると：

- $X$ から $\mathbb{N}$ への全単射は存在しない
- 濃度は $|X| > \aleph_0$

__具体例__

- $\mathbb{R}$（実数全体）：**非可算**  
  - カントールの対角線論法により、$\mathbb{N} \to \mathbb{R}$ の全単射は存在しないことが示される
  - 濃度は**連続体濃度** $\mathfrak{c}$ と呼ばれ、$\mathfrak{c} > \aleph_0$
- $\mathcal{P}(\mathbb{N})$（自然数のべき集合）：**非可算**  
  - Cantorの定理により $|\mathcal{P}(\mathbb{N})| > |\mathbb{N}| = \aleph_0$
  - 実際には $|\mathcal{P}(\mathbb{N})| = 2^{\aleph_0} = \mathfrak{c}$
- 区間 $(0,1)$ や $[0,1]$ も $\mathbb{R}$ と対等なので非可算

### 3. 可算と非可算の性質の違い

__(1) 部分集合との関係__
- 可算集合の任意の無限部分集合は、**可算無限**または有限
  - 例：偶数全体、素数全体はどちらも $\mathbb{N}$ と対等
- 非可算集合には、**非可算な部分集合**が必ず存在する
  - 例：$\mathbb{R}$ の任意の区間 $(a,b)$ は非可算

__(2) 和集合・直積の振る舞い__
- 可算集合の**有限和・有限直積**は可算集合
  - 例：$\mathbb{Z} \times \mathbb{Z}$（整数の格子）は可算無限
- 可算集合の**可算無限個の和集合**も可算集合（可算和は可算）
  - 例：可算無限個の有限集合の和集合は可算
- 非可算集合との和集合・直積は、多くの場合非可算
  - 例：$\mathbb{R} \times \mathbb{R}$ は $\mathbb{R}$ と対等（濃度 $\mathfrak{c}$）

__(3) 濃度の比較__
- 可算集合の濃度：$\aleph_0$（最小の無限濃度）
- 非可算集合の濃度：$\aleph_0$ より真に大きい  
  - 例：$\mathfrak{c} = 2^{\aleph_0} > \aleph_0$


### 4. なぜこの区別が重要か

- **可算集合**は「要素を1,2,3,…と数え上げられる」という意味で、**扱いやすい無限**です。
- **非可算集合**は「自然数では数え上げられないほど多い」無限で、実数・関数空間・べき集合など、より複雑な対象に対応します。
- この区別により、「どのくらいの無限か」を厳密に議論でき、解析学・位相空間論・確率論など多くの分野で本質的な役割を果たします。

3. 自然数全体 $\mathbb{N}$ と整数全体 $\mathbb{Z}$

**問題8（Cantorの定理の理解）**  
Cantorの定理により、任意の集合 $X$ に対して $|X| < |\mathcal{P}(X)|$ が成り立つ。

1. $X$ が有限集合のとき、この不等式が成り立つことを直接確認せよ。
2. $X = \mathbb{N}$ のとき、$\mathcal{P}(\mathbb{N})$ が非可算であることを、Cantorの定理を用いて説明せよ。
3. $\mathcal{P}(\mathbb{N})$ と $\mathbb{R}$ が同じ濃度を持つことを、Cantorの定理と他の事実を用いて説明せよ。


**問題9（対角線論法の練習）**

- $X = \mathbb{N}$ とする。
- 全射 $f: \mathbb{N} \to \mathcal{P}(\mathbb{N})$ が存在すると仮定する。
- このとき、集合 $D = \{n \in \mathbb{N} \mid n \notin f(n)\}$ を構成し、  
  任意の $m \in \mathbb{N}$ に対して $f(m) \neq D$ となることを示せ。

**問題10（総合）**  
次の問いに答えよ。

1. 有限集合の真部分集合は、必ず元の集合より小さい濃度を持つことを示せ。
2. 無限集合には、必ず真部分集合であって同じ濃度を持つものが存在することを示せ。
3. 可算無限集合 $A$ と非可算集合 $B$ があるとき、$|A| < |B|$ であることを示せ。
4. 任意の集合 $X$ に対して、$|X| < |\mathcal{P}(X)|$ が成り立つことを、Cantorの定理を用いて説明せよ。

### 解答

以下、各問題の解答です。

__第1部：対等・濃度の定義と具体例__

__問題1（対等の判定）__

1. **$A = \{1,2,3\},\ B = \{a,b,c\}$**  
   - 全単射 $f(1)=a,\ f(2)=b,\ f(3)=c$ が存在するので、**対等**。

2. **$A = \mathbb{N},\ B = \{2,4,6,\dots\}$**  
   - 全単射 $f(n) = 2n$ が存在するので、**対等**（どちらも可算無限）。

3. **$A = \mathbb{Z},\ B = \mathbb{Q}$**  
   - $\mathbb{Z}$ も $\mathbb{Q}$ も可算無限集合であり、自然数との全単射が構成できるので、**対等**。

4. **$A = \mathbb{R},\ B = (0,1)$**  
   - 全単射 $f(x) = \frac{1}{\pi}\arctan x + \frac{1}{2}$ などにより、$\mathbb{R} \sim (0,1)$ が示せるので、**対等**（どちらも連続体濃度）。

5. **$A = \mathbb{N},\ B = \mathcal{P}(\mathbb{N})$**  
   - Cantorの定理により $|\mathbb{N}| < |\mathcal{P}(\mathbb{N})|$ なので、**対等ではない**。

__問題2（濃度の比較）__

1. **$A = \{1,2,3\},\ B = \{a,b\}$**  
   - $|A| = 3,\ |B| = 2$ なので、**$|A| > |B|$**。

2. **$A = \mathbb{N},\ B = \mathbb{Z}$**  
   - どちらも可算無限で、全単射が構成できるので、**$|A| = |B|$**。

3. **$A = \mathbb{Q},\ B = \mathbb{R}$**  
   - $\mathbb{Q}$ は可算無限、$\mathbb{R}$ は非可算（連続体濃度）なので、**$|A| < |B|$**。

4. **$A = \mathbb{R},\ B = \mathcal{P}(\mathbb{R})$**  
   - Cantorの定理により $|\mathbb{R}| < |\mathcal{P}(\mathbb{R})|$ なので、**$|A| < |B|$**。

5. **$A = \mathbb{N},\ B = \mathbb{N} \times \mathbb{N}$**  
   - 全単射 $(m,n) \mapsto 2^m 3^n$ などにより、$\mathbb{N} \times \mathbb{N} \sim \mathbb{N}$ が示せるので、**$|A| = |B|$**。

__第2部：有限集合・無限集合の判定と性質__

__問題3（有限・無限の判定）__

1. **$A = \{n \in \mathbb{N} \mid n^2 < 10\}$**  
   - $n = 1,2,3$ のみが条件を満たすので、**有限集合**（要素数 3）。

2. **$B = \{x \in \mathbb{R} \mid x^2 = 2\}$**  
   - $x = \sqrt{2}, -\sqrt{2}$ の2点のみなので、**有限集合**（要素数 2）。

3. **$C = \{(m,n) \in \mathbb{Z} \times \mathbb{Z} \mid m^2 + n^2 \leq 10\}$**  
   - $m,n$ は有限範囲（$|m|,|n| \leq 3$ 程度）に限られるので、**有限集合**。

4. **$D = \{ \frac{1}{n} \mid n \in \mathbb{N} \}$**  
   - 自然数 $n$ と1対1に対応するので、**無限集合**（可算無限）。

5. **$E = \{A \subset \mathbb{N} \mid |A| = 1\}$**  
   - 各1点集合 $\{n\}$ と自然数 $n$ が1対1に対応するので、**無限集合**（可算無限）。

__問題4（無限集合の特徴づけ）__

> 命題：$X$ が無限集合 $\iff$ $X$ から $X$ 自身への**単射であって全射でない写像**が存在する。

1. **$X = \mathbb{N}$**  
   - 写像 $f(n) = n+1$ は単射だが、1 に移る元がないので全射ではない。  
   - よって $\mathbb{N}$ は無限集合。

2. **$X = \mathbb{Z}$**  
   - 写像 $f(n) = n+1$ は単射だが、0 に移る元がないので全射ではない。  
   - よって $\mathbb{Z}$ は無限集合。

3. **$X = \mathbb{R}$**  
   - 写像 $f(x) = x+1$ は単射だが、0 に移る元がないので全射ではない。  
   - よって $\mathbb{R}$ は無限集合。

__第3部：可算集合・非可算集合__

__問題5（可算・非可算の判定）__

1. **$A = \mathbb{N} \times \mathbb{N}$**  
   - 全単射 $(m,n) \mapsto 2^m 3^n$ などにより $\mathbb{N} \times \mathbb{N} \sim \mathbb{N}$ なので、**可算集合**（可算無限）。

2. **$B = \mathbb{Q} \times \mathbb{Q}$**  
   - $\mathbb{Q}$ が可算なので、その有限直積も可算 → **可算集合**。

3. **$C = \{0,1\}^{\mathbb{N}}$**  
   - 自然数上の二進列全体であり、$\mathcal{P}(\mathbb{N})$ と対等。  
   - Cantorの定理により非可算 → **非可算集合**。

4. **$D = \{A \subset \mathbb{N} \mid A \text{は有限集合}\}$**  
   - 有限部分集合全体は、自然数と有限列の対応で可算 → **可算集合**。

5. **$E = \{A \subset \mathbb{N} \mid A \text{は無限集合}\}$**  
   - $\mathcal{P}(\mathbb{N})$ から有限部分集合全体（可算）を除いたもの。  
   - 非可算集合から可算集合を除いても非可算 → **非可算集合**。

__問題6（可算和・可算直積の性質）__

1. **可算無限個の有限集合の和集合は、常に可算集合である。**  
   - **正しい**。  
   - 各有限集合の要素を列挙し、自然数と1対1に対応づけることで可算集合になる。

2. **可算無限個の可算無限集合の直積は、常に可算集合である。**  
   - **誤り**。  
   - 反例：$\{0,1\}^{\mathbb{N}}$（可算無限個の2点集合の直積）は非可算。

3. **可算無限個の非可算集合の和集合は、常に非可算集合である。**  
   - **誤り**。  
   - 反例：各 $A_n = [n, n+1]$ は非可算だが、和集合 $\bigcup_n A_n = [0,\infty)$ は非可算とは限らない？  
     実際には $[0,\infty)$ も非可算なので、より適切な反例は「互いに素でない非可算集合の和集合が可算になる例」を考える必要があるが、標準的な反例は少ない。  
     一般には「常に非可算」とは限らない（例：各 $A_n$ が同じ非可算集合なら和集合も非可算だが、配置によっては可算になる可能性もある）。

__第4部：Cantor–Bernsteinの定理とCantorの定理の適用__

__問題7（Cantor–Bernsteinの定理の適用）__

1. **$(0,1)$ と $[0,1]$**  
   - 包含写像 $(0,1) \hookrightarrow [0,1]$ は単射 → $|(0,1)| \leq |[0,1]|$。  
   - 写像 $[0,1] \to (0,1),\ x \mapsto \frac{x}{3} + \frac{1}{3}$ は単射 → $|[0,1]| \leq |(0,1)|$。  
   - Cantor–Bernsteinの定理より $|(0,1)| = |[0,1]|$。

2. **$\mathbb{R}$ と $\mathbb{R}^2$**  
   - 包含写像 $\mathbb{R} \hookrightarrow \mathbb{R}^2,\ x \mapsto (x,0)$ は単射 → $|\mathbb{R}| \leq |\mathbb{R}^2|$。  
   - 空間充填曲数（例：Peano曲線）などにより、$\mathbb{R} \to \mathbb{R}^2$ の全射（したがって単射も構成可能）が存在 → $|\mathbb{R}^2| \leq |\mathbb{R}|$。  
   - Cantor–Bernsteinの定理より $|\mathbb{R}| = |\mathbb{R}^2|$。

3. **$\mathbb{N}$ と $\mathbb{Z}$**  
   - 包含写像 $\mathbb{N} \hookrightarrow \mathbb{Z}$ は単射 → $|\mathbb{N}| \leq |\mathbb{Z}|$。  
   - 写像 $\mathbb{Z} \to \mathbb{N},\ n \mapsto 2|n| + [n>0]$ などは単射 → $|\mathbb{Z}| \leq |\mathbb{N}|$。  
   - Cantor–Bernsteinの定理より $|\mathbb{N}| = |\mathbb{Z}|$。

__問題8（Cantorの定理の理解）__

1. **$X$ が有限集合のとき、$|X| < |\mathcal{P}(X)|$**  
   - $|X| = n$ なら $|\mathcal{P}(X)| = 2^n$。  
   - 任意の自然数 $n$ について $n < 2^n$ が成り立つので、$|X| < |\mathcal{P}(X)|$。

2. **$X = \mathbb{N}$ のとき、$\mathcal{P}(\mathbb{N})$ が非可算であること**  
   - Cantorの定理により $|\mathbb{N}| < |\mathcal{P}(\mathbb{N})|$。  
   - $|\mathbb{N}| = \aleph_0$（可算無限）なので、$|\mathcal{P}(\mathbb{N})| > \aleph_0$、すなわち非可算。

3. **$\mathcal{P}(\mathbb{N})$ と $\mathbb{R}$ が同じ濃度を持つこと**  
   - $\mathcal{P}(\mathbb{N})$ は二進列 $\{0,1\}^{\mathbb{N}}$ と対等であり、各二進列は実数の二進展開（適切に修正）と対応づけられる。  
   - これにより $|\mathcal{P}(\mathbb{N})| = |\{0,1\}^{\mathbb{N}}| = |\mathbb{R}|$（連続体濃度 $\mathfrak{c}$）。

__問題9（対角線論法の練習）__

- $X = \mathbb{N}$、全射 $f: \mathbb{N} \to \mathcal{P}(\mathbb{N})$ が存在すると仮定。
- 集合 $D = \{n \in \mathbb{N} \mid n \notin f(n)\}$ を構成。
- 任意の $m \in \mathbb{N}$ について：
  - $m \in D \iff m \notin f(m)$。
  - もし $f(m) = D$ なら、$m \in D \iff m \notin D$ となり矛盾。
  - よって任意の $m$ に対し $f(m) \neq D$。
- したがって $D \in \mathcal{P}(\mathbb{N})$ は $f$ の像に含まれず、$f$ は全射ではない → 矛盾。  
  よって全射 $f: \mathbb{N} \to \mathcal{P}(\mathbb{N})$ は存在しない。

__第5部：総合問題__

__問題10（総合）__

1. **有限集合の真部分集合は、必ず元の集合より小さい濃度を持つ**  
   - 有限集合 $A$ の真部分集合 $B \subsetneq A$ は、要素数が $|B| < |A|$。  
   - 濃度の定義（全単射の存在）により、$|B| < |A|$。

2. **無限集合には、必ず真部分集合であって同じ濃度を持つものが存在する**  
   - 無限集合 $X$ は、ある真部分集合 $Y \subsetneq X$ と全単射 $f: X \to Y$ を持つ（Dedekind無限の定義）。  
   - よって $|X| = |Y|$。

3. **可算無限集合 $A$ と非可算集合 $B$ があるとき、$|A| < |B|$**  
   - $A$ は可算無限なので $|A| = \aleph_0$。  
   - $B$ は非可算なので $|B| > \aleph_0$。  
   - よって $|A| < |B|$。

4. **任意の集合 $X$ に対して、$|X| < |\mathcal{P}(X)|$**  
   - Cantorの定理により、全射 $X \to \mathcal{P}(X)$ は存在しない。  
   - 一方、包含写像 $x \mapsto \{x\}$ は単射 $X \to \mathcal{P}(X)$ を与えるので $|X| \leq |\mathcal{P}(X)|$。  
   - 全射がないため $|X| \neq |\mathcal{P}(X)|$、したがって $|X| < |\mathcal{P}(X)|$。
