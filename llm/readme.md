LLM（大規模言語モデル）の主要学会について示す。
LLMは、自然言語処理（NLP）や機械学習（ML）の一分野として扱われることが多いため、関連する主要な学会は主に以下のカテゴリに分かれます。


## 🐟 自然言語処理・計算言語学系（LLMの中心的な学会）

### ACL（Association for Computational Linguistics）
- **国際計算言語学会**。NLP分野で最も権威のある国際学会の一つです。
- 年次総会（ACL）やEMNLP（Empirical Methods in Natural Language Processing）など、LLM関連の論文が多数発表されます。
- 特に **EMNLP** は、LLMの実証的研究（事前学習、ファインチューニング、評価など）が集まる場として重要です。

### NAACL（North American Chapter of the ACL）
- ACLの北米支部の年次会議。北米を中心に、LLM・NLPの最新研究が発表されます。

### EACL（European Chapter of the ACL）
- ACLの欧州支部の年次会議。欧州のNLPコミュニティの中心的な場です。


## 🐠 機械学習・AI全般系（LLMの基盤技術）

### NeurIPS（Neural Information Processing Systems）
- **NeurIPS** は、機械学習・深層学習分野で最も権威のある国際会議の一つです。
- LLMのアーキテクチャ、学習手法、理論解析、マルチモーダルなど、基盤技術に関する論文が多数発表されます。

### ICML（International Conference on Machine Learning）
- 機械学習分野のトップ会議の一つ。LLMの学習アルゴリズム、最適化、スケーリング則などがよく扱われます。

### ICLR（International Conference on Learning Representations）
- 表現学習を中心とした深層学習の国際会議。
- Transformer系モデルやLLMの表現学習・事前学習に関する研究が多く、LLMの理論・手法面で重要な場です。


## 🐡 AI・AI応用系（LLMの応用・倫理・社会実装）

### AAAI（Association for the Advancement of Artificial Intelligence）
- AI全般の国際会議。LLMの応用（対話システム、要約、コード生成など）やAI倫理・安全性に関する研究も扱われます。

### IJCAI（International Joint Conference on Artificial Intelligence）
- もう一つのAI全般の国際会議。LLMを含むAI技術の幅広い応用が議論されます。

## 🐬 まとめ

LLMに特化した「LLM学会」という名称の学会は（少なくとも2024年時点では）ありませんが、**ACL系（ACL / EMNLP / NAACL / EACL）** と **機械学習トップ会議（NeurIPS / ICML / ICLR）** が、LLM研究の中心的な場として位置づけられています。

- **モデル設計・学習手法・理論** に強い関心がある方：NeurIPS / ICML / ICLR
- **自然言語処理・LLM応用・評価** に強い関心がある方：ACL / EMNLP / NAACL / EACL

といった形で、関心領域に応じてフォーカスする学会を選ぶのが一般的です。
