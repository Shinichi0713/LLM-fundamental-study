**サブワード（subword）**とは、単語をさらに小さな意味のまとまりに分割した単位のことです。

---

### ✅ 背景

自然言語処理で使うモデルは、文章を**トークン（数値化単位）**に変換して処理します。

しかし、未知語（辞書にない単語）が出てくると、従来の単語単位の辞書だけでは対応できません。

そこで登場するのが**サブワード**です。

---

### ✅ サブワードの例

| 単語                   | サブワード分割例   |
| ---------------------- | ------------------ |
| **Transformer**  | Trans + former     |
| **DeBERTa**      | De + BERT + a      |
| **機械学習**     | 機械 + 学習        |
| **自然言語処理** | 自然 + 言語 + 処理 |

* 頻出部分は1つのサブワードとして扱う
* 未知語は既知のサブワードに分解して扱う

---

### ✅ サブワードを使うメリット

1. **未知語への対応**
   * 「機械学習モデル」が辞書にない場合でも

     → 「機械学習」「モデル」に分解して処理可能
2. **語彙サイズを抑えられる**
   * 単語辞書だと膨大になるが、サブワードなら数万単位で済む
3. **多言語対応が容易**
   * 共通の文字単位で分割できるため、新しい言語や漢字にも対応

---

### ✅ まとめ

> **サブワード = 単語より小さく、文字より大きい「意味を持つ最小単位」**
>
> モデルはこれを使って文章を効率的に理解・処理する
