## カルマンフィルタとは

ざっくりいうと以下です。

> 「状態空間モデル（SSM）を前提に、
> 予測と観測を組み合わせて、
> ノイズを含む観測から真の状態を推定するアルゴリズム」

* **予測** ：モデルに基づいた将来の状態
* **更新** ：観測値で予測を修正
* **結果** ：ノイズに強い、滑らかな状態推定

ということでカルマンフィルタを仕込んだシステムに対して入出力、カルマンフィルタの用途は以下です。

* **入力** ：ノイズを含む観測値（例：GPS の位置、センサーの測定値）
* **出力** ：真の状態の推定値（例：実際の位置、速度）
* **カルマンフィルタの用途** ：観測ノイズを除去し、**最も確からしい状態**を推定する

### 1, 前提：状態空間モデル（SSM）

カルマンフィルタは、**状態空間モデル（SSM）** を前提とします。

* **状態方程式** ：状態がどう変化するか

$$
$${ x_t = A x_{t-1} + B u_t + w_t }
$$

* **観測方程式** ：状態から観測がどう生成されるか

$$
$${ z_t = H x_t + v_t }
$$

* $$
  { x_t }$$：真の状態（位置・速度など）
  $$
* $$
  { z_t }$$：観測値（センサー測定値）
  $$
* $$
  { w_t, v_t }$$：ノイズ
  $$

### 2. アルゴリズムの流れ（2 ステップ）

カルマンフィルタで行っていることは、事前に予測しておいた状態推定値とその不確実性を、実際の観測 

$$
{ z_t }
$$

 を使って修正しています。


* 予測ステップで「モデルベースの推定」を行い、
* 観測更新ステップで「実際の観測とのズレ」を**カルマンゲインで重み付けして補正**し、
* 同時に「状態推定の不確実性」も更新する

処理のイメージは以下のようになります。観測値と**現在のモデルの線形推定値の誤差を最小化するためのカルマンゲインを、観測値の更新の度に、合わせて更新していく**とみてください。

![](https://assets.st-note.com/img/1777853906-Jab7y3EG1dXzcuxFCVgPZTmi.png)
Kalman filter process

**(1) 予測ステップ（Predict）**

* 前の状態から**次の状態の予測**を行う

$$
{  \hat{x}_{t|t-1} = A \hat{x}_{t-1|t-1} + B u_t }
$$

$$
{ P_{t|t-1} = A P_{t-1|t-1} A^\top + Q }
$$

* $$
  { \hat{x}_{t|t-1} }$$：予測された状態
  $$
* $$
  { P_{t|t-1} }$$：予測誤差の共分散
  $$
* $$
  { Q }$$：  **プロセスノイズ共分散** 。状態方程式における外乱・モデル誤差の大きさと相関を表す。
  $$

**(2) 更新ステップ（Update）**

* 実際の観測 
  $$
  { z_t }
  $$

   を使って、**予測を修正**する

$$
{ K_t = P_{t|t-1} H^\top (H P_{t|t-1} H^\top + R)^{-1} }
$$

$$
{ \hat{x}{t|t} = \hat{x}_{t|t-1} + K_t (z_t - H \hat{x}_{t|t-1} )  }
$$

$$
{ P_{t|t} = (I - K_t H) P_{t|t-1} }
$$

* $$
  { K_t }$$：カルマンゲイン（観測をどれだけ信じるか）
  $$
* $$
  { \hat{x}_{t|t} }$$：修正後の状態推定値
  $$
* $$
  { H }$$: 観測モデル行列。**状態から観測への線形写像**を表す行列。
  $$

### 3. 直感的なイメージ

* **予測ステップ** ：
  「前の状態から、次はこう動くはず」と予測
* **更新ステップ** ：
  「実際の観測値と予測の差を見て、予測を修正」

 **例** ：車の位置推定

* 予測：速度から「1 秒後はここにいるはず」
* 観測：GPS が「ここにいる」と報告
* カルマンフィルタ：両方を重み付けして、**最も確からしい位置**を計算

## 提案の沿革

* **カルマンフィルタは「状態空間モデル（SSM）を前提とした最適フィルタ」として設計されました。**
* 1960 年に Rudolf E. Kálmán が発表した論文では、 **状態方程式と観測方程式（SSM）を明示的に使って** 、最適推定アルゴリズムを導出しています。

### 1. カルマンフィルタの起源（1960 年）

**カルマンの論文**

* 1960 年、Rudolf E. Kálmán が **"A New Approach to Linear Filtering and Prediction Problems"** を発表[IEEE Xplore](https://ieeexplore.ieee.org/document/1098349)。
* この論文で、**状態空間モデル（SSM）を前提とした最適フィルタ**が提案されました。

**状態空間モデルの明示的使用**

* 論文内では、以下のような**状態方程式と観測方程式**が定義されています。
  * 状態方程式：
    $$
    { x_{k+1} = \Phi_k x_k + G_k u_k + w_k }
    $$
  * 観測方程式：
    $$
    { z_k = H_k x_k + v_k }
    $$
* ここで、
  * $$
    { x_k }$$：状態ベクトル
    $$
  * $$
    { z_k }$$：観測ベクトル
    $$
  * $$
    { \Phi_k, G_k, H_k }$$：システム行列
    $$
  * $$
    { w_k, v_k }$$：ノイズ（白色ガウスノイズ）
    $$

**つまり、カルマンフィルタは「SSM を前提にした最適推定アルゴリズム」として生まれました。**

### 2. SSM 自体の歴史（カルマン以前）

**状態空間モデルの起源**

* SSM の原型は、**制御理論やシステム理論**で古くから使われていました。
* 例：
  * 1920–30 年代：線形システムの微分方程式モデル
  * 1940–50 年代：制御理論（ウィーナーフィルタなど）

**ウィーナーフィルタとの違い**

* ウィーナーフィルタ（1940 年代）は、**周波数領域**で最適フィルタを設計します。
* カルマンフィルタは、**状態空間（時間領域）**で最適フィルタを設計します。
* カルマンの貢献は、**SSM を明示的に使い、時間領域で再帰的な最適推定アルゴリズムを導出した**点にあります。

## 具体例

ラジコンなどに搭載されているGPSを用いて自己位置推定する場合に使われています。

### SSM の定義

* 状態 
  $$
  { x_t = [位置, 速度]^\top }
  $$
* 観測 
  $$
  { z_t = GPS の位置測定値 }
  $$

**状態方程式**

$$
{ x_t = \begin{bmatrix} 1 & \Delta t \ 0 & 1 \end{bmatrix} x_{t-1} + w_t }
$$

* $$
  { \Delta t }$$：サンプリング間隔
  $$
* $$
  { w_t }$$：加速度などの外乱（ノイズ）
  $$

**観測方程式**

$$
{ z_t = \begin{bmatrix} 1 & 0 \end{bmatrix} x_t + v_t }
$$

* $$
  { v_t }$$：GPS の測定誤差（ノイズ）
  $$

### カルマンフィルタの役割

* GPS の測定値 
  $$
  { z_t }
  $$

   はノイズを含むため、 **そのままでは真の位置ではない** 。
* SSM に基づいて「車はこのくらい動くはず」という**予測**と、
  GPS の**観測**を組み合わせ、**最適な推定値**を計算する。

## 例題

カルマンフィルタを使って、**ノイズを含む観測から真の値を予測する例題**を案出しました。
ここでは、**1次元の位置**を推定するシンプルな例を Python で実装します。

### 設定

* 真の位置：等速直線運動（速度 1.0）
* 観測：真の位置にノイズを加えたもの
* カルマンフィルタ：状態 = [位置, 速度]、観測 = 位置

### 実装コード

```
import numpy as np
import matplotlib.pyplot as plt

# カルマンフィルタの実装
class KalmanFilter:
    def __init__(self, F, H, Q, R, x0, P0):
        self.F = F  # 状態遷移行列
        self.H = H  # 観測行列
        self.Q = Q  # プロセスノイズ共分散
        self.R = R  # 観測ノイズ共分散
        self.x = x0 # 状態推定値
        self.P = P0 # 誤差共分散

    def predict(self):
        # 予測ステップ
        self.x = self.F @ self.x
        self.P = self.F @ self.P @ self.F.T + self.Q

    def update(self, z):
        # 更新ステップ
        y = z - self.H @ self.x  # イノベーション（観測と予測の差）
        S = self.H @ self.P @ self.H.T + self.R  # イノベーション共分散
        K = self.P @ self.H.T @ np.linalg.inv(S)  # カルマンゲイン

        self.x = self.x + K @ y
        self.P = (np.eye(2) - K @ self.H) @ self.P

# パラメータ設定
dt = 1.0  # サンプリング間隔
F = np.array([[1, dt],
              [0, 1]])  # 状態遷移行列: 等速運動
H = np.array([[1, 0]])  # 観測行列: 位置のみ観測
Q = np.array([[0.1, 0],
              [0, 0.1]])  # プロセスノイズ（小さめ）
R = np.array([[1.0]])     # 観測ノイズ（大きめ）

# 初期状態
x0 = np.array([0, 1.0])  # [位置, 速度]
P0 = np.eye(2) * 1.0    # 初期誤差共分散

# カルマンフィルタのインスタンス
kf = KalmanFilter(F, H, Q, R, x0, P0)

# データ生成
T = 50
true_positions = np.zeros(T)
observed_positions = np.zeros(T)
estimated_positions = np.zeros(T)

# 真の位置: 等速運動
for t in range(T):
    true_positions[t] = t * 1.0  # 速度 1.0

# 観測: 真の位置 + ノイズ
np.random.seed(42)
observed_positions = true_positions + np.random.normal(0, 1.0, T)

# カルマンフィルタの実行
for t in range(T):
    # 予測ステップ
    kf.predict()
    # 更新ステップ（観測値を用いて修正）
    kf.update(observed_positions[t:t+1])
    # 推定された位置を保存
    estimated_positions[t] = kf.x[0]

# 可視化
plt.figure(figsize=(10, 6))
plt.plot(true_positions, label='真の位置', linestyle='--', color='black')
plt.plot(observed_positions, label='観測値（ノイズあり）', marker='o', markersize=3, alpha=0.7)
plt.plot(estimated_positions, label='カルマンフィルタ推定値', linewidth=2)
plt.title('カルマンフィルタによる位置推定')
plt.xlabel('時間ステップ')
plt.ylabel('位置')
plt.legend()
plt.grid(True)
plt.show()
```

### 実行結果の解釈

![](https://assets.st-note.com/img/1777785755-EXYls97Ug21QdTGSzh08IbCf.png)
Kalman filter processing result

**1. 真の位置 vs 観測値**

* 真の位置：直線（等速運動）
* 観測値：真の位置にノイズが乗ったもの（ばらつきあり）

**2. カルマンフィルタの推定値**

* 観測値よりも**滑らかで真の位置に近い軌道**を描きます。
* これは、カルマンフィルタが
  * **モデル（等速運動）に基づいた予測**と
  * **観測値**を組み合わせて、**最適な推定値**を計算しているためです。

**3. ノイズ除去の効果**

* 観測値のノイズが **大幅に減り** 、真の位置に近づいています。
* 特に、**急激な変動（ノイズによるジャンプ）が平滑化**されていることがわかります。

## 総括

* **カルマンフィルタ**は、ノイズを含む観測データから、システムの真の状態（位置・速度など）を推定するアルゴリズムです。
* その前提として、**状態空間モデル（SSM）** を用います。
  * 状態方程式：状態がどう変化するか（

    $$
    { x_t = A x_{t-1} + B u_t + w_t }
    $$

    ）
  * 観測方程式：状態から観測がどう生成されるか（

    $$
    { z_t = H x_t + v_t }
    $$

    ）
* SSM が「状態の動き方」を定義し、カルマンフィルタが「その状態をどう推定するか」を実現します。
* **SSM**は、状態の遷移と観測の生成を定義する数学モデルであり、カルマンフィルタは SSM を前提に真の状態を推定します。
* 近年の深層学習（S4, Mamba など）では、この  **SSM の考え方をニューラルネットワークに組み込み** 、長距離依存を扱うシーケンスモデルとして再利用しています。
  * SSM：過去の情報を内部状態に要約
  * カルマンフィルタ：その状態を最適に推定
  * Mamba：選択的 SSM で表現力を高め、線形時間で計算
